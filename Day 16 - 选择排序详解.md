### 💡 选择排序的定义

**官方说法：**
> 选择排序是一种简单直观的排序算法，它的工作原理是每一次从待排序的数据元素中选出最小（或最大）的一个元素，存放在序列的起始位置

**人话版：**
> **选择排序 = 每次挑出最大的，放到该放的位置**

```javascript

function selectionSort(arr) {
    const n = arr.length;
    
    console.log('原始数组:', arr);
    console.log(`开始排序，共${n}个元素\n`);
    
    // 外层循环：控制已排序区域的边界
    for (let i = 0; i < n - 1; i++) {
        console.log(`\n=== 第 ${i + 1} 轮 ===`);
        console.log(`当前数组：`, arr);
        
        // 假设当前位置是最小的
        let minIndex = i;
        
        // 内层循环：在未排序区域找真正的最小值
        console.log(`  从位置${i}开始找最小值...`);
        
        for (let j = i + 1; j < n; j++) {
            console.log(`    比较位置${j}: ${arr[j]} (当前最小：${arr[minIndex]})`);
            
            if (arr[j] < arr[minIndex]) {
                minIndex = j;
                console.log(`      发现更小的！更新最小索引为${j}`);
            }
        }
        
        // 如果找到更小的，就交换
        if (minIndex !== i) {
            console.log(`  交换位置${i}和${minIndex}: ${arr[i]} ↔ ${arr[minIndex]}`);
            [arr[i], arr[minIndex]] = [arr[minIndex], arr[i]];
            console.log(`  交换后：`, arr);
        } else {
            console.log(`  最小值已在正确位置，无需交换`);
        }
        
        console.log(`第${i + 1}轮结果：`, arr);
        console.log(`  ↑ 前${i + 1}个元素已有序`);
    }
    
    console.log('\n✅ 排序完成:', arr);
    return arr;
}
```

### 简洁版本（实用）

```javascript
/**
 * 选择排序 - 简洁版（实际使用）
 */

function selectionSortSimple(arr) {
    const n = arr.length;
    
    for (let i = 0; i < n - 1; i++) {
        let minIndex = i;
        
        // 找最小值的索引
        for (let j = i + 1; j < n; j++) {
            if (arr[j] < arr[minIndex]) {
                minIndex = j;
            }
        }
        
        // 交换
        if (minIndex !== i) {
            [arr[i], arr[minIndex]] = [arr[minIndex], arr[i]];
        }
    }
    
    return arr;
}

// 测试
console.log(selectionSortSimple([64, 34, 25, 12, 22, 11, 90]));
// [11, 12, 22, 25, 34, 64, 90]

console.log(selectionSortSimple([5, 1, 4, 2, 8]));
// [1, 2, 4, 5, 8]
```

## ⚖️ 第 4 步：选择排序 vs 冒泡排序（20 分钟）

### 详细对比

```javascript
/**
 * 冒泡排序 vs 选择排序
 * 
 * 相同点：
 * ✓ 都是简单排序算法
 * ✓ 时间复杂度都是 O(n²)
 * ✓ 空间复杂度都是 O(1)
 * ✓ 都适合小规模数据
 * 
 * 不同点：
 */

// 1. 操作方式不同
冒泡排序：
→ 相邻元素比较
→ 大的往后交换
→ 每轮最大的"浮"到最后

选择排序：
→ 扫描找最小值
→ 直接交换到前面
→ 每轮最小的"选"到前面


// 2. 交换次数不同（关键区别！）
冒泡排序：
→ 最坏情况：O(n²) 次交换
→ 每次比较都可能交换

选择排序：
→ 最多 O(n) 次交换
→ 每轮最多交换 1 次

例子：[5, 4, 3, 2, 1]

冒泡排序需要交换：
第 1 轮：交换 4 次 → [4, 3, 2, 1, 5]
第 2 轮：交换 3 次 → [3, 2, 1, 4, 5]
第 3 轮：交换 2 次 → [2, 1, 3, 4, 5]
第 4 轮：交换 1 次 → [1, 2, 3, 4, 5]
总共：10 次交换 ❌

选择排序：
第 1 轮：交换 1 次 → [1, 4, 3, 2, 5]
第 2 轮：交换 1 次 → [1, 2, 3, 4, 5]
第 3 轮：交换 1 次 → [1, 2, 3, 4, 5]
第 4 轮：交换 0 次 → [1, 2, 3, 4, 5]
总共：3 次交换 ✅


// 3. 稳定性不同
冒泡排序：稳定 ✅
→ 相同元素不会改变相对位置

选择排序：不稳定 ❌
→ 可能会改变相同元素的相对位置

例子：[5a, 5b, 2]
选择排序第 1 轮：
找到 2，和 5a 交换
[2, 5b, 5a]
↑
5a 和 5b 的顺序变了！❌


// 4. 性能表现
最好情况（已经有序）：
冒泡排序：O(n) ← 可以提前结束 ✅
选择排序：O(n²) ← 还是要扫描所有 ❌

最坏情况（完全逆序）：
冒泡排序：O(n²)
选择排序：O(n²)

平均情况：
冒泡排序：O(n²)
选择排序：O(n²)
```

---

### 对比表格

| 特性 | 冒泡排序 | 选择排序 |
|-----|---------|---------|
| **比较次数** | O(n²) | O(n²) |
| **交换次数** | O(n²) | O(n) ✅ |
| **最好情况** | O(n) ✅ | O(n²) |
| **最坏情况** | O(n²) | O(n²) |
| **稳定性** | 稳定 ✅ | 不稳定 |
| **实现难度** | 简单 | 简单 |
| **适用场景** | 小规模<br>基本有序 | 小规模<br>交换代价大 |

```javascript
// 冒泡排序
function bubbleSort(arr) {
    for (let i = 0; i < arr.length - 1; i++) {
        for (let j = 0; j < arr.length - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                // 频繁交换 ❌
                [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]];
            }
        }
    }
}

// 选择排序
function selectionSort(arr) {
    for (let i = 0; i < arr.length - 1; i++) {
        let minIndex = i;
        for (let j = i + 1; j < arr.length; j++) {
            if (arr[j] < arr[minIndex]) {
                minIndex = j;
            }
        }
        // 每轮只交换 1 次 ✅
        if (minIndex !== i) {
            [arr[i], arr[minIndex]] = [arr[minIndex], arr[i]];
        }
    }
}
```

## 📊 第 5 步：性能分析（20 分钟）

### 时间复杂度分析

```javascript
/**
 * 选择排序的时间复杂度：
 * 
 * 无论什么情况下，都要扫描所有元素找最小值
 * 所以永远是 O(n²)
 */

// 比较次数计算

第 1 轮：比较 n-1 次
第 2 轮：比较 n-2 次
第 3 轮：比较 n-3 次
...
第 n-1 轮：比较 1 次

总比较次数 = (n-1) + (n-2) + ... + 1
           = n(n-1)/2
           = O(n²)

// 交换次数

最好情况（已经有序）：0 次
最坏情况（完全逆序）：n-1 次
平均情况：O(n)

结论：
✓ 比较次数固定，无法优化
✓ 交换次数远少于冒泡排序
✓ 适合交换代价大的场景
```

---

### 空间复杂度

```javascript
/**
 * 空间复杂度：O(1)
 * 
 * 只需要常数个额外变量：
 * - minIndex
 * - i, j 循环变量
 * 
 * 原地排序，不需要额外数组
 */
```

---

### 优缺点总结

```javascript
/**
 * 优点：
 * ✓ 实现简单
 * ✓ 交换次数少（最多 n-1 次）
 * ✓ 不需要额外空间
 * ✓ 适合小数据量
 * 
 * 缺点：
 * ✗ 效率低，O(n²)
 * ✗ 不稳定
 * ✗ 无法利用数据的部分有序性
 * ✗ 即使已经有序，也要完整执行
 */

// 适用场景

✅ 适合：
- 数据量小（n < 50）
- 交换操作代价大
- 对稳定性无要求
- 简单应用，不追求极致性能

❌ 不适合：
- 大数据量
- 要求稳定性
- 数据基本有序
- 高性能要求
```

## 💻 第 6 步：实战项目（30 分钟）

### 项目：比赛选手排名系统

```javascript
/**
 * 歌唱比赛评分系统
 * 
 * 功能：
 * 1. 录入选手得分
 * 2. 按总分排序（选择排序）
 * 3. 显示最终排名
 * 4. 统计奖项分布
 */

class Contestant {
    constructor(id, name, scores) {
        this.id = id;              // 编号
        this.name = name;          // 姓名
        this.scores = scores;      // 评委打分数组
        this.total = 0;            // 总分
        this.average = 0;          // 平均分
        this.rank = 0;             // 排名
        this.calculateScore();
    }
    
    // 计算得分（去掉最高分和最低分，取平均）
    calculateScore() {
        const sorted = [...this.scores].sort((a, b) => a - b);
        const validScores = sorted.slice(1, -1);  // 去掉最高最低
        this.total = validScores.reduce((sum, s) => sum + s, 0);
        this.average = this.total / validScores.length;
    }
    
    toString() {
        return `${this.name}(总分:${this.average.toFixed(2)})`;
    }
}

class CompetitionSystem {
    constructor(name) {
        this.name = name;
        this.contestants = [];
    }
    
    // 添加选手
    addContestant(id, name, scores) {
        const contestant = new Contestant(id, name, scores);
        this.contestants.push(contestant);
        console.log(`✅ 添加选手：${name}, 得分：${contestant.average.toFixed(2)}`);
    }
    
    // 按得分排序（选择排序）
    sortByScore() {
        console.log('\n🏆 开始按得分排序...\n');
        
        const arr = this.contestants;
        const n = arr.length;
        
        for (let i = 0; i < n - 1; i++) {
            let maxIndex = i;
            
            // 找剩余选手中得分最高的
            for (let j = i + 1; j < n; j++) {
                if (arr[j].average > arr[maxIndex].average) {
                    maxIndex = j;
                }
            }
            
            // 交换
            if (maxIndex !== i) {
                [arr[i], arr[maxIndex]] = [arr[maxIndex], arr[i]];
            }
        }
        
        // 设置排名
        this.updateRanks();
        
        console.log('✅ 排序完成！\n');
    }
    
    // 更新排名
    updateRanks() {
        let rank = 1;
        this.contestants[0].rank = rank;
        
        for (let i = 1; i < this.contestants.length; i++) {
            if (this.contestants[i].average < this.contestants[i - 1].average) {
                rank = i + 1;
            }
            this.contestants[i].rank = rank;
        }
    }
    
    // 显示最终排名
    showRankings() {
        console.log('\n╔════════════════════════════════════════════╗');
        console.log(`║   ${this.name} - 最终排名                   ║`);
        console.log('╚════════════════════════════════════════════╝\n');
        
        console.log('排名\t编号\t姓名\t得分\t奖项');
        console.log('─'.repeat(50));
        
        for (let c of this.contestants) {
            const medal = c.rank <= 3 ? ['🥇', '🥈', '🥉'][c.rank - 1] : '  ';
            const award = this.getAward(c.rank);
            console.log(`${medal} ${c.rank}\t${c.id}\t${c.name}\t${c.average.toFixed(2)}\t${award}`);
        }
        
        console.log();
    }
    
    // 获取奖项名称
    getAward(rank) {
        if (rank === 1) return '🏆 金奖';
        if (rank === 2) return '🥈 银奖';
        if (rank === 3) return '🥉 铜奖';
        if (rank <= 5) return '优秀奖';
        return '鼓励奖';
    }
    
    // 统计奖项分布
    countAwards() {
        console.log('\n📊 奖项统计：\n');
        
        const awards = {
            gold: 0,    // 金奖
            silver: 0,  // 银奖
            bronze: 0,  // 铜奖
            excellence: 0,  // 优秀奖
            encouragement: 0  // 鼓励奖
        };
        
        for (let c of this.contestants) {
            const rank = c.rank;
            if (rank === 1) awards.gold++;
            else if (rank === 2) awards.silver++;
            else if (rank === 3) awards.bronze++;
            else if (rank <= 5) awards.excellence++;

        }
```

### ✅ 你今天学到了：

**1. 选择排序原理**
```
✓ 每次选择最小的

```

```

✓ 适合小数据
```

```
✓ 简单排名
✓ 交换代价大的场景
```
