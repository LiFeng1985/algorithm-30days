
```
周日晚上：
- 填写周总结
- 分析规律
- 制定下周计划
```

```
月底：

```

```
- 爆款标准：> 10,000

成长期（第2-3个月）：
- 单篇平均：2000-5000
- 爆款标准：> 20,000

成熟期（第4-6个月）：
- 单篇平均：5000-10,000
- 爆款标准：> 50,000
```

### 转化率基准
```
点赞率：
- < 3%：需要优化内容
- 3-5%：正常
- 5-8%：良好
- > 8%：优秀

收藏率：
- < 5%：需要优化内容
- 5-10%：正常
- 10-15%：良好
- > 15%：优秀

评论率：
- < 1%：需要加强互动
- 1-2%：正常
- 2-5%：良好
- > 5%：优秀
```
