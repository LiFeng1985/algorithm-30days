## 📌 中心主题：图的遍历

```
                    ┌─────────────┐
                    │  图的遍历   │
                    └──────┬──────┘
                           │
              ┌────────────┴────────────┐
              │                         │
          ┌───┴───┐                 ┌───┴───┐
          │  BFS  │                 │  DFS  │
          └───┬───┘                 └───┬───┘
```

---

## 分支 1：BFS（广度优先搜索）

```
BFS
    │
    ├─ 实现方式
    │   └─ 队列（FIFO）
    │
    ├─ 遍历顺序
    │   └─ 按层遍历
    │       └─ 先近后远
    │
    ├─ 特点
    │   ├─ 保证最短路径（无权图）
    │   ├─ 需要较多内存
    │   └─ 时间复杂度 O(V+E)
    │
    └─ 应用
        ├─ 最短路径
        ├─ 层次遍历
        └─ 社交网络分析
```

---

## 分支 2：DFS（深度优先搜索）

```
DFS
    │
    ├─ 实现方式
    │   ├─ 递归（系统栈）
    │   └─ 迭代（显式栈）
    │
    ├─ 遍历顺序
    │   └─ 深入到底再回溯
    │
    ├─ 特点
    │   ├─ 内存使用少
    │   ├─ 不保证最短路径
    │   └─ 时间复杂度 O(V+E)
    │
    └─ 应用
        ├─ 检测环
        ├─ 拓扑排序
        └─ 连通性检查
```

---

## 分支 3：对比总结

```
对比
    │
    ├─ 数据结构
    │   ├─ BFS: 队列
    │   └─ DFS: 栈
    │
    ├─ 遍历策略
    │   ├─ BFS: 宽度优先
    │   └─ DFS: 深度优先
    │
    ├─ 最短路径
    │   ├─ BFS: 保证
    │   └─ DFS: 不保证
    │
    └─ 内存使用
        ├─ BFS: 较多
        └─ DFS: 较少
```

---

## 分支 4：关键要点

```
要点
    │
    ├─ visited集合
    │   └─ 避免重复访问和死循环
    │
    ├─ 时间复杂度
    │   └─ 都是 O(V+E)
    │
    ├─ 空间复杂度
    │   └─ 都是 O(V)
    │
    └─ 选择原则
        ├─ 找最短路径 → BFS
        ├─ 检测环 → DFS
        └─ 其他看需求
```

---

## 🔑 核心要点

### 两个关键词
```
1️⃣ BFS
   队列 + 按层遍历

2️⃣ DFS
   栈/递归 + 深入回溯
```

### 一个中心思想
```
💡 系统化探索
不遗漏地访问所有可达节点
```

### 重要技巧
```
✓ 必须标记已访问
✓ 根据需求选择算法
✓ 理解各自的特点
✓ 掌握实现方法
```

---

## 💡 学习建议

### 理解重点
```
1. 掌握两种遍历的原理
2. 理解为什么用不同的数据结构
3. 明白各自的优缺点
4. 学会选择合适的算法
```

### 练习方向
```
✓ 能手写BFS
✓ 能手写DFS（递归和迭代）
✓ 能用遍历解决问题
✓ 能分析复杂度
```

### 常见误区
```
✗ 忘记标记已访问
✗ BFS用栈
✗ 认为DFS能找到最短路径
✗ 不理解适用场景
```

## 📊 遍历算法选择指南

```

   ↓ 是 → BFS

   ↓ 是 → DFS

3. 需要拓扑排序？
   ↓ 是 → DFS

4. 内存紧张？

   ↓ 根据具体需求选择
```
