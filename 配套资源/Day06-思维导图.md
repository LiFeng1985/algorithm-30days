## 分支 2：递归的两个要素

```
递归的要素
    │
    ├─ 基准情况 ✓
    │   ├─ 终止条件
    │   ├─ 最简单的情况
    │   └─ 防止无限递归
    │
    └─ 递归情况 ✓
        ├─ 调用自己
        ├─ 参数更接近基准
        └─ 分解成子问题
```

## 🔑 核心要点

### 三个关键词
```
1️⃣ 自己调用自己
   函数的递归调用

2️⃣ 基准情况
   必须有终止条件

3️⃣ 分解
   把大问题变小
```

### 一个中心思想
```
💡 大事化小，小事化了
复杂问题简单化
```
