### 题目 2：递归的两个要素（15 分）

**参考答案：**

**要素 1：基准情况（Base Case）**（7 分）
```
作用：递归的终止条件
```

```
作用：把问题分解成更小的子问题

通过解决子问题来解决原问题
```

## 二、代码实践题答案

### 题目 4：计算阶乘（20 分）

**参考答案：**

```javascript
function factorial(n) {
    // 基准情况
    if (n === 0 || n === 1) {
        return 1;
    }
    
    // 递归调用
    return n * factorial(n - 1);
}

// 测试
console.log(factorial(0));  // 1
console.log(factorial(1));  // 1
console.log(factorial(5));  // 120
```

**执行过程：**
```
factorial(5)
= 5 * factorial(4)
= 5 * (4 * factorial(3))
= 5 * (4 * (3 * factorial(2)))
= 5 * (4 * (3 * (2 * factorial(1))))
= 5 * (4 * (3 * (2 * 1)))
= 5 * (4 * (3 * 2))
= 5 * (4 * 6)
= 5 * 24
= 120
```

**评分要点：**
- 基准情况正确（6 分）
- 递归调用正确（10 分）
- 能通过测试（4 分）

### 题目 5：斐波那契数列（20 分）

```javascript
function fibonacci(n) {
    // 基准情况
    if (n === 1 || n === 2) {
        return 1;
    }
    
    // 递归调用
    return fibonacci(n - 1) + fibonacci(n - 2);
}

// 测试
console.log(fibonacci(1));  // 1
console.log(fibonacci(2));  // 1
console.log(fibonacci(10)); // 55
```

### 题目 6：递归求和（15 分）

**参考答案：**

```javascript
function sumArray(arr) {
    // 基准情况：空数组
    if (arr.length === 0) {
        return 0;
    }
    
    // 递归调用：第一个元素 + 剩余元素的和
    return arr[0] + sumArray(arr.slice(1));
}

// 测试
console.log(sumArray([1, 2, 3, 4, 5]));  // 15
console.log(sumArray([]));               // 0
```

**评分要点：**
- 基准情况正确（5 分）
- 递归逻辑正确（7 分）
- 能通过测试（3 分）

### 题目 8：汉诺塔问题（10 分）

**参考答案：**

```
汉诺塔的递归思路：

要把 n 个盘子从 A 移到 C：

1. 先把上面 n-1 个盘子从 A 移到 B（借助 C）
2. 把最大的盘子从 A 移到 C
3. 再把 n-1 个盘子从 B 移到 C（借助 A）

这样就把大问题分解成了小问题：
- 移动 n 个盘子 → 移动 n-1 个盘子 + 移动 1 个盘子 + 移动 n-1 个盘子
- 一直分解到只有 1 个盘子，直接移动就行
```

**评分要点：**
- 说清楚三步走策略（6 分）
- 提到分解思想（2 分）
- 提到基准情况（2 分）

```
你可以一个一个数，

每堆再分成两半，
最后数最小的几堆，加起来就行。

所以，递归就是把大问题变成小问题的方法！

谢谢大家！
```
