### 题目 1：什么是插入排序？（10 分）

**参考答案：**
```
插入排序是一种简单的排序算法，
通过构建有序序列，
将未排序元素逐个插入到已排序部分的正确位置。
就像整理扑克牌一样自然。
```

### 题目 2：插入排序的原理（15 分）

**参考答案：**

**基本思想：**（6 分）
```
将数组分为已排序和未排序两部分。
初始时第一个元素为已排序部分。
每轮取未排序部分的第一个元素，
在已排序部分找到合适位置并插入。
```

**为什么像整理扑克牌：**（4 分）
```
就像我们摸牌时，
每摸一张新牌，
就会把它插到手中牌的合适位置，
这样手中的牌始终是有序的。
```

**执行过程：**（5 分）
```
第一轮：第二个元素插入到前面
第二轮：第三个元素插入到前两个中
...
第 n-1 轮：完成排序
```

**评分要点：**
- 基本思想 6 分
- 扑克牌比喻 4 分
- 执行过程 5 分

### 题目 3：复杂度分析（10 分）

**参考答案：**

```
时间复杂度：
最好情况：O(n) - 已经有序，只需比较
平均情况：O(n²)
最坏情况：O(n²) - 完全逆序，需要大量移动

空间复杂度：O(1) - 原地排序

稳定性：稳定 - 相等元素不会改变相对位置
```

**评分要点：**
- 最好情况 O(n) 得 2 分
- 最坏/平均 O(n²) 得 2 分
- 空间复杂度 O(1) 得 3 分
- 稳定性正确得 3 分

**解释：**
```
为什么插入排序稳定？
因为遇到相等元素时停止移动，
所以相等元素的相对位置不会改变。
例如：[5, 8, 5, 2, 9]
插入第二个 5 时，会在第一个 5 后面停下
```

## 二、代码实践题答案

### 题目 4：实现基础插入排序（20 分）

**参考答案：**

```javascript
function insertionSort(arr) {
    const n = arr.length;
    
    // 从第二个元素开始（索引 1）
    for (let i = 1; i < n; i++) {
        const current = arr[i];  // 待插入的元素
        let j = i - 1;  // 已排序部分的最后一个
        
        // 在已排序部分找插入位置
        while (j >= 0 && arr[j] > current) {
            arr[j + 1] = arr[j];  // 向后移动
            j--;
        }
        
        // 插入到正确位置
        arr[j + 1] = current;
    }
    
    return arr;
}

// 测试
console.log(insertionSort([64, 34, 25, 12, 22, 11, 90]));
// [11, 12, 22, 25, 34, 64, 90] ✓

console.log(insertionSort([5, 1, 4, 2, 8]));
// [1, 2, 4, 5, 8] ✓
```

**执行过程演示：**
```
初始：[5, 1, 4, 2, 8]

第一轮（i=1）：
current = 1
j = 0, arr[0]=5 > 1 → 移动：[5, 5, 4, 2, 8]
j = -1, 退出循环
arr[0] = 1 → [1, 5, 4, 2, 8]

第二轮（i=2）：
current = 4
j = 1, arr[1]=5 > 4 → 移动：[1, 5, 5, 2, 8]
j = 0, arr[0]=1 < 4 → 停止
arr[1] = 4 → [1, 4, 5, 2, 8]

第三轮（i=3）：
current = 2
j = 2, arr[2]=5 > 2 → 移动：[1, 4, 5, 5, 8]
j = 1, arr[1]=4 > 2 → 移动：[1, 4, 4, 5, 8]
j = 0, arr[0]=1 < 2 → 停止
arr[1] = 2 → [1, 2, 4, 5, 8]

第四轮（i=4）：
current = 8
j = 3, arr[3]=5 < 8 → 不移动
arr[4] = 8 → [1, 2, 4, 5, 8]

最终结果：[1, 2, 4, 5, 8] ✓
```

**评分要点：**
- 外层循环正确（4 分）
- 保存当前元素（3 分）
- while 循环条件正确（5 分）
- 移动元素正确（4 分）
- 插入到正确位置（2 分）
- 能通过测试（2 分）

**常见错误：**
❌ 外层循环从 0 开始 → ✅ 应该从 1 开始
❌ while 条件用 || → ✅ 应该用 &&
❌ 忘记 arr[j + 1] = current → ✅ 最后要插入

### 题目 5：对比三种简单排序（25 分）

**参考答案：**

```javascript
function compareThreeSorts() {
    const testCases = [
        [64, 34, 25, 12, 22, 11, 90],
        [1, 2, 3, 4, 5, 6, 7],  // 已经有序
        [7, 6, 5, 4, 3, 2, 1],  // 完全逆序
        [5, 5, 5, 5, 5],         // 全部相同
        [3]                       // 单个元素
    ];
    
    for (const arr of testCases) {
        console.log('\n测试数据:', arr);
        
        // 测试冒泡排序
        const start1 = performance.now();
        bubbleSort([...arr]);
        const time1 = performance.now() - start1;
        console.log('冒泡排序耗时:', time1.toFixed(2), 'ms');
        
        // 测试选择排序
        const start2 = performance.now();
        selectionSort([...arr]);
        const time2 = performance.now() - start2;
        console.log('选择排序耗时:', time2.toFixed(2), 'ms');
        
        // 测试插入排序
        const start3 = performance.now();
        insertionSort([...arr]);
        const time3 = performance.now() - start3;
        console.log('插入排序耗时:', time3.toFixed(2), 'ms');
        
        // 找出最快的
        const minTime = Math.min(time1, time2, time3);
        if (minTime === time1) console.log('最快：冒泡排序');
        else if (minTime === time2) console.log('最快：选择排序');
        else console.log('最快：插入排序');
    }
}

// 辅助函数（略，见题目）

compareThreeSorts();
```

### 题目 7：插入排序的应用场景（10 分）

**参考答案：**

**适用场景：**（4 分）
```
1. 数据量很小的情况（n < 50）
2. 基本已经有序的数据
```

```
✓ 原地排序，空间复杂度 O(1)
```

```
✗ 即使发现有序也要继续比较
```

### 题目 8：改进思路（10 分）

**参考答案：**

**思路 1：二分插入排序**
```
使用二分查找找到插入位置，
减少比较次数从 O(n) 到 O(log n)。
但移动次数不变。

预期效果：
总时间复杂度仍为 O(n²)
但比较次数减少
```

**思路 2：希尔排序**
```
先对间隔较大的子序列排序，
逐步缩小间隔，最后做直接插入排序。

预期效果：
大幅提升效率
时间复杂度可达 O(n^1.3)
```

**评分要点：**
- 提出合理思路 6 分
- 说明预期效果 4 分
