## 🔑 核心要点

### 三个关键词
```
1️⃣ 哈希函数
   键 → 索引的转换

2️⃣ 冲突处理
   多个键映射到同一位置

3️⃣ 负载因子
   已用空间 / 总空间
```

### 一个中心思想
```
💡 空间换时间
用额外空间换取 O(1) 查找
```

### 重要技巧
```
✓ 选择合适的哈希函数
✓ 控制负载因子
✓ 及时处理冲突
✓ 适时扩容
```

## 💡 学习建议

### 理解重点
```
1. 掌握哈希表的基本原理
2. 理解冲突处理的两种方法
3. 明白为什么平均是 O(1)
4. 学会使用 Map 和 Object
```

### 练习方向
```
✓ 能手写简单哈希表
✓ 能用 Map 解决问题
✓ 能分析时间和空间复杂度
✓ 能识别适用场景
```

### 常见误区
```
✗ 认为查找一定是 O(1)
✗ 忽略冲突处理
✗ 不控制负载因子
✗ 混淆 Map 和 Object
```

