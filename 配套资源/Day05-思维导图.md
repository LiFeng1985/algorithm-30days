## 分支 1：队列是什么？

```
队列是什么
    │
    ├─ 定义
    │   ├─ FIFO（先进先出）
    │   ├─ 操作受限的线性表
    │   └─ 队尾插入，队头删除
    │
    ├─ 生活例子
    │   ├─ 排队买票
    │   ├─ 打印任务
    │   └─ 击鼓传花
    │
    └─ 术语
        ├─ 队头（Front）
        ├─ 队尾（Rear）
        └─ 空队列
```

---

## 分支 2：队列的特点

```
队列的特点
    │
    ├─ 优点 ✓
    │   ├─ 操作简单 O(1)
    │   ├─ 公平性（FIFO）
    │   └─ 应用广泛
    │
    └─ 缺点 ✗
        ├─ 只能访问队头
        ├─ 不支持随机访问
        └─ 容量可能受限
```

---

## 分支 3：基本操作

```
基本操作（都是 O(1)）
    │
    ├─ enqueue(element)
    │   └─ 入队（队尾）
    │
    ├─ dequeue()
    │   └─ 出队（队头）
    │
    ├─ front()
    │   └─ 查看队头
    │
    ├─ isEmpty()
    │   └─ 判空
    │
    └─ size()
        └─ 获取大小
```

---

## 🔑 核心要点

### 三个关键词
```
1️⃣ FIFO
   先进先出

2️⃣ 公平
   先来后到，人人平等

3️⃣ 层次
   BFS 按层次遍历的基础
```

### 一个中心思想
```
💡 顺序决定公平
FIFO 保证了处理的公平性
```
