## 💡 学习建议

### 成功要素
```
1. 明确的目标
2. 可行的计划
3. 持续的执行力
4. 良好的学习习惯
5. 积极的心态
```

### 避免的陷阱
```
✗ 急于求成
✗ 三天打鱼两天晒网
✗ 只看不练
✗ 不总结反思
✗ 孤立学习
```

### 推荐的节奏
```
每天：
- 30-60 分钟学习
- 至少 1 道算法题
- 简短总结

每周：
- 5-10 小时深度学习
- 完成一个小目标
- 写一篇博客或笔记

每月：
- 回顾 progress
- 调整计划
- 设定新目标
```
