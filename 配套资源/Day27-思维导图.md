## 分支 3：与 DP 对比

```
对比
    │
    ├─ 相同点
    │   ├─ 都用于最优化
    │   └─ 都有最优子结构
    │
    ├─ 不同点
    │   ├─ 最优保证
    │   │   ├─ 贪心：不一定
    │   │   └─ DP: 保证
    │   ├─ 决策方式
    │   │   ├─ 贪心：自顶向下
    │   │   └─ DP: 自底向上
    │   └─ 复杂度
    │       ├─ 贪心：O(n) 或 O(n log n)
    │       └─ DP: O(n²) 或更高
    │
    └─ 选择原则
        ├─ 能用贪心优先贪心
        └─ 否则用 DP
```

## 🔑 核心要点

### 三个关键词
```
1️⃣ 当前最优
   只看眼前，不管未来

2️⃣ 贪心选择
   做出选择后不后悔

3️⃣ 证明
   必须证明贪心能得到最优
```

### 一个中心思想
```
💡 短视策略
通过一系列局部最优选择
期望达到全局最优
```

### 重要技巧
```
✓ 先尝试贪心（简单高效）
✓ 不行再用 DP（保证最优）
✓ 一定要证明正确性
✓ 注意适用范围
```

## 💡 学习建议

### 理解重点
```
1. 掌握贪心的基本思想
2. 理解贪心选择性质
3. 明白为什么不一定最优
4. 学会证明贪心正确性
```

### 练习方向
```
✓ 能实现经典贪心问题
✓ 能判断问题是否适合贪心
✓ 能证明贪心正确性
✓ 能举出贪心失败的例子
```

### 常见误区
```
✗ 认为贪心总是最优
✗ 不证明就用贪心
✗ 混淆贪心和 DP
✗ 忽略适用条件
```

## 📊 贪心问题识别指南

```
遇到一个问题，如何判断能否用贪心？

1. 是否是最优化问题？
   ↓ 是 → 可能用贪心

2. 是否有贪心选择性质？
   （当前最优选择能导致全局最优）
   ↓ 是 → 可能用贪心

3. 是否能证明贪心正确？
   （数学归纳法或反证法）
   ↓ 是 → 可以用贪心！

4. 时间要求高吗？
   ↓ 是 → 优先贪心

5. 需要精确最优吗？
   ↓ 否 → 可以用贪心
   
经典贪心问题：
✓ 活动选择
✓ 分数背包
✓ 霍夫曼编码
✓ Dijkstra 最短路径
✓ 最小生成树
```
