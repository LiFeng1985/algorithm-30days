## 🔑 第二周核心要点

### 四个重要数据结构
```
1️⃣ 树
   层次结构，递归定义

2️⃣ 堆
   完全二叉树，优先队列

3️⃣ 哈希表
   键值对，快速查找

4️⃣ 图
   节点和边，关系建模
```

### 三个重要算法
```
1️⃣ BFS
   队列实现，找最短路径

2️⃣ DFS
   栈/递归，深入探索

3️⃣ Dijkstra
   贪心策略，加权最短路径
```

### 一个中心思想
```
💡 选择合适的工具
不同场景用不同的数据结构
```
