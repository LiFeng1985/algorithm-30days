## 🔑 核心要点

### 三个关键词
```
1️⃣ 分治
   分解→解决→合并

2️⃣ 合并
   两个有序数组合并成一个

3️⃣ 稳定
   始终 O(n log n)，稳定排序
```

### 一个中心思想
```
💡 分而治之
把大问题拆成小问题，各个击破
```

### 重要技巧
```
✓ merge 时用 <= 保证稳定性
✓ 处理剩余元素用 concat
✓ 递归基准情况是单元素
```

## 💡 学习建议

### 理解重点
```
1. 先理解分治法的思想
2. 掌握 merge 函数的实现
3. 理解为什么时间复杂度是 O(n log n)
4. 明白为什么需要 O(n) 空间
```

### 练习方向
```
✓ 手写 merge 函数
✓ 手写 mergeSort 递归
✓ 手动模拟归并过程
✓ 分析时间和空间复杂度
```

### 常见误区
```
✗ 忘记处理剩余元素
✗ 用 < 而不是 <=（影响稳定性）
✗ 修改原数组而不是创建新数组
✗ 基准情况不完整
```
