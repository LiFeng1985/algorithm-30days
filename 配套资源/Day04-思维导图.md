## 分支 1：栈是什么？

```
栈是什么
    │
    ├─ 定义
    │   ├─ LIFO（后进先出）
    │   ├─ 操作受限的线性表
    │   └─ 只能在一端操作
    │
    ├─ 生活例子
    │   ├─ 一摞盘子
    │   ├─ 羽毛球筒
    │   └─ 浏览器后退
    │
    └─ 术语
        ├─ 栈顶（Top）
        ├─ 栈底（Bottom）
        └─ 空栈
```

---

## 分支 2：栈的特点

```
栈的特点
    │
    ├─ 优点 ✓
    │   ├─ 操作简单 O(1)
    │   ├─ 实现容易
    │   └─ 应用广泛
    │
    └─ 缺点 ✗
        ├─ 只能访问栈顶
        ├─ 不支持随机访问
        └─ 容量可能受限
```

## 🔑 核心要点

### 三个关键词
```
1️⃣ LIFO
   后进先出

2️⃣ 栈顶
   所有操作都在这一端

3️⃣ 简单
   实现和应用都很简单
```

### 一个中心思想
```
💡 操作受限 = 效率更高
限制带来效率和便利
```
