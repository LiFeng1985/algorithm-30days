## 一、基础概念题答案

### 题目 1：什么是队列？（10 分）

**参考答案：**
```
队列是一种先进先出（FIFO）的数据结构，
就像排队买票，先来的人先买到。
只能从队尾添加，从队头删除。
```

### 题目 2：FIFO 的含义（10 分）

**参考答案：**

**含义：** First In First Out，先进先出
```
最先进入队列的元素，第一个被拿出来。
就像排队买奶茶，先到的人先买到，
后到的人要等前面的人都买完。
```

**生活例子：**
- 排队买票：先来的人先买
- 打印机任务：先提交的任务先打印
- 食堂打饭：排在前面的先打饭

**评分要点：**
- 解释清楚 FIFO（5 分）
- 例子恰当（5 分）

## 二、代码实践题答案

### 题目 4：实现队列类（25 分）

**参考答案：**

```javascript
class Queue {
    constructor() {
        this.items = [];
    }
    
    enqueue(element) {
        this.items.push(element);
    }
    
    dequeue() {
        if (this.isEmpty()) {
            return null;
        }
        return this.items.shift();
    }
    
    front() {
        if (this.isEmpty()) {
            return null;
        }
        return this.items[0];
    }
    
    isEmpty() {
        return this.items.length === 0;
    }
    
    size() {
        return this.items.length;
    }
}
```

**评分要点：**
- 构造函数正确（3 分）

- front 正确（6 分，含边界检查 2 分）

### 题目 5：击鼓传花游戏（20 分）

**参考答案：**

```javascript
function hotPotato(names, num) {
    const queue = new Queue();
    
    // 所有人入队
    for (let name of names) {
        queue.enqueue(name);
    }
    
    // 模拟传递过程
    while (queue.size() > 1) {
        // 传递 num 次
        for (let i = 0; i < num; i++) {
            const person = queue.dequeue();
            queue.enqueue(person);
        }
        // 淘汰一个人
        queue.dequeue();
    }
    
    return queue.dequeue();  // 返回获胜者
}
```

**评分要点：**
- 使用队列存储人员（4 分）
- 正确的循环条件（4 分）
- 传递逻辑正确（6 分）
- 淘汰逻辑正确（4 分）
- 返回获胜者（2 分）

### 题目 6：打印任务调度（15 分）

**参考答案：**

```javascript
function printTasks(tasks) {
    const queue = new Queue();
    const completed = [];
    
    // 所有任务入队
    for (let task of tasks) {
        queue.enqueue(task);
    }
    
    // 按顺序处理
    while (!queue.isEmpty()) {
        const task = queue.dequeue();
        console.log(`正在打印：${task}`);
        completed.push(task);
    }
    
    return completed;
}
```

**评分要点：**
- 使用队列存储任务（3 分）
- 正确的入队操作（3 分）
- 正确的出队操作（3 分）

### 题目 8：BFS 为什么用队列？（10 分）

**参考答案：**

```
BFS 要用队列是因为：

1. 需要按层次遍历
   队列的 FIFO 特性保证先访问的节点先处理

2. 保证遍历顺序
   同一层的节点按顺序入队，下一层按顺序访问

3. 符合 BFS 的思想
   从起点开始，一层一层往外扩展

4. 简单高效
   队列操作简单，时间复杂度 O(1)
```

**评分要点：**
- 提到按层次遍历（3 分）
- 提到 FIFO 特性（3 分）
- 提到遍历顺序（2 分）
- 提到效率（2 分）
