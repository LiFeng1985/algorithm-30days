## 🔑 核心要点

### 八大数据结构
```
1️⃣ 数组 - 基础中的基础
2️⃣ 链表 - 灵活的数据组织
3️⃣ 栈 - LIFO 的经典
4️⃣ 队列 - FIFO 的代表
5️⃣ 树 - 层次结构
6️⃣ 堆 - 优先队列
7️⃣ 哈希表 - 快速查找
8️⃣ 图 - 复杂关系
```

### 六大排序算法
```
简单排序：
- 冒泡：两两比较交换
- 选择：每次选最小
- 插入：逐个插入有序区

高效排序：
- 归并：分治 + 合并
- 快速：分区 + 递归
- 堆排序：建堆 + 调整
```

### 四大算法思想
```
1️⃣ 动态规划 - 记住子问题的解
2️⃣ 贪心算法 - 每一步选最优
3️⃣ 回溯算法 - 试错 + 回退
4️⃣ 分治算法 - 分而治之
```
