## 分支 1：基本概念

```
概念
    │
    ├─ 顶点（Vertex）
    │   └─ 图中的节点
    │
    ├─ 边（Edge）
    │   └─ 连接两个顶点的线
    │
    ├─ 度（Degree）
    │   ├─ 无向图：邻居数量
    │   └─ 有向图：入度 + 出度
    │
    └─ 路径（Path）
        └─ 顶点序列
```

---

## 分支 2：表示方法

```
表示
    │
    ├─ 邻接矩阵
    │   ├─ 二维数组
    │   ├─ 空间 O(V²)
    │   ├─ 查询快 O(1)
    │   └─ 适合稠密图
    │
    └─ 邻接表
        ├─ 数组/Map + 链表
        ├─ 空间 O(V+E)
        ├─ 遍历快 O(度)
        └─ 适合稀疏图
```

## 🔑 核心要点

### 三个关键词
```
1️⃣ 顶点
   图中的节点

2️⃣ 边
   节点之间的关系

3️⃣ 连通性
   节点之间是否可达
```

### 一个中心思想
```
💡 关系建模
用图表示事物之间的复杂关系
```

### 重要技巧
```
✓ 选择合适的表示方法
✓ 理解有向和无向的区别
✓ 掌握基本操作
✓ 为后续算法打基础
```

---

## 💡 学习建议

### 理解重点
```
1. 掌握图的基本概念
2. 理解两种表示方法
3. 明白何时使用哪种
4. 学会实现基本操作
```

### 练习方向
```
✓ 能手写图的类
✓ 能用邻接表表示图
✓ 能添加顶点和边
✓ 能分析时间和空间复杂度
```

### 常见误区
```
✗ 混淆有向和无向
✗ 选错表示方法
✗ 忘记处理边界
✗ 不理解度的概念
```

## 📊 图的使用指南

```
何时使用图？

1. 需要表示关系？
   ↓ 是 → 考虑图

2. 关系复杂（多对多）？
   ↓ 是 → 考虑图

3. 需要找路径或连通性？
   ↓ 是 → 必须用图！

经典应用：
✓ 社交网络
✓ 地图导航
✓ 任务调度
✓ 网络路由
✓ 依赖管理
```
