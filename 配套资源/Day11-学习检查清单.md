- [ ] 哈希表的查找一定是 O(1)
- [ ] 哈希函数可以把任意键映射到数组索引
- [ ] 负载因子越大，性能越好
- [ ] JavaScript 的 Object 就是哈希表

**答案：**
1. ❌（最坏情况是 O(n)，当所有键都冲突时）
2. ✅（但可能有冲突）
3. ❌（负载因子越大，冲突越多，性能越差）
4. ✅（但有一些限制，如键必须是字符串或 Symbol）
