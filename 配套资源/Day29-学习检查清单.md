## 📚 知识点自查

### 数据结构回顾
```
□ 数组：连续存储，随机访问
□ 链表：节点链接，灵活插入删除
□ 栈：后进先出（LIFO）
□ 队列：先进先出（FIFO）
□ 树：层次结构，递归定义
□ 堆：完全二叉树，优先队列
□ 哈希表：键值对，快速查找
□ 图：节点和边，网络结构
```

### 算法回顾
```
□ 排序算法：冒泡、选择、插入、归并、快速、堆排序
□ 查找算法：线性查找、二分查找
□ 遍历算法：BFS、DFS
□ 最优化算法：动态规划、贪心、回溯
□ 其他：递归、分治
```

### 复杂度分析
```
□ 能计算常见算法的时间复杂度
□ 能计算常见算法的空间复杂度
□ 理解 O(1)、O(log n)、O(n)、O(n log n)、O(n²)、O(2^n)
□ 知道如何优化复杂度
```
