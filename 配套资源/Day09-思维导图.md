```
                    ┌─────────────┐
                    │  二叉搜索树  │
                    └──────┬──────┘
                           │
      ┌──────────┬─────────┴─────────┬──────────┐
```

---

## 分支 1：BST 的定义

```
BST 定义
    │
    ├─ 基本性质
    │   ├─ 左子树 < 根节点
    │   └─ 右子树 > 根节点
    │
    ├─ 重要特点
    │   ├─ 中序遍历 = 升序
    │   └─ 左右子树也是 BST
    │
    └─ 生活例子
        ├─ 猜数字游戏
        └─ 整理扑克牌
```

---

## 分支 2：BST 的操作

```
BST 操作
    │
    ├─ 查找 Search
    │   ├─ 小于往左找
    │   └─ 大于往右找
    │
    ├─ 插入 Insert
    │   ├─ 找到合适位置
    │   └─ 创建新节点
    │
    ├─ 删除 Delete
    │   ├─ 叶节点：直接删
    │   ├─ 一个子节点：继承
    │   └─ 两个子节点：找后继
    │
    └─ 验证 Valid
        └─ 检查范围限制
```

---

## 分支 3：时间复杂度

```
时间复杂度
    │
    ├─ 最好情况
    │   └─ O(1) - 第一次就找到
    │
    ├─ 平均情况
    │   └─ O(log n) - 平衡树
    │
    └─ 最坏情况
        └─ O(n) - 退化为链表
        
关键：树的平衡性
```

---

## 分支 4：BST 的应用

```
BST 应用
    │
    ├─ 数据库索引
    │   └─ 快速查找记录
    │
    ├─ Map/Set实现
    │   └─ 有序键值对
    │
    ├─ 文件系统
    │   └─ 目录索引
    │
    └─ 编译器
        └─ 符号表
```

---

## 🔑 核心要点

### 三个关键词
```
1️⃣ 左小右大
   左边小，右边大

2️⃣ 二分
   每次排除一半

3️⃣ 中序升序
   中序遍历得到升序
```

### 一个中心思想
```
💡 有序带来高效
利用大小关系快速定位
```
