## 💡 提示与启发

### 解题思路提示

**题目 4：**
- 添加元素用 `push()`
- 删除元素用 `pop()`

**题目 6：**
- 从头到尾遍历数组
- 逐个比较是否相等

**题目 9：**
- 方法 1：用 Set 自动去重
- 方法 2：新建数组，只添加没出现过的元素

**题目 10：**
- 暴力法：两层循环尝试所有组合
- 优化：用 Map 记录已遍历的数字

### 常见错误提醒

❌ 数组索引从 1 开始 → ✅ JavaScript 索引从 0 开始
❌ 直接修改原数组 → ✅ 注意是否需要保持原数组不变
❌ 忘记边界检查 → ✅ 检查索引是否越界
