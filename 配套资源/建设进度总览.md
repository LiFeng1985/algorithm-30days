## 📋 资源配置标准

### 每种资源的行数标准

| 资源类型 | 标准行数 | 说明 |
|---------|---------|------|
| 学习检查清单 | 200-250 行 | 目标明确、可勾选、自测 |
| 练习题 | 300-400 行 | 10-12 道题，分层次 |
| 答案详解 | 450-550 行 | 详细解析、评分标准 |
| 思维导图 | 350-400 行 | ASCII 图形、完整框架 |
| **合计** | **1,300-1,600 行** | **每天配套资源** |

### 内容结构标准

**学习检查清单：**
```
□ 学习目标检查（知识/能力/思维）
□ 知识点自查
□ 自我测试题
□ 实践能力检查
□ 时间分配建议
□ 掌握程度自评
□ 完成认证
```

**练习题：**
```
一、基础概念题（必做）
   - 概念理解
   - 特点说明
   - 简答解释
   
二、代码实践题（必做）
   - 基本操作
   - 函数实现
   - 调试改错
   
三、理解应用题（选做）
   - 应用场景
   - 对比分析
   - 方案设计
   
四、拓展挑战题（挑战题）
   - LeetCode 风格
   - 综合应用
   - 创新思维
   
五、费曼输出（必做）
   - 小老师环节
```

**答案详解：**
```
✓ 每道题的标准答案
✓ 评分要点和分值分配
✓ 优秀答案示例
✓ 常见错误分析
✓ 改进建议
✓ 下一步学习指导
```

**思维导图：**
```
中心主题
├─ 分支 1：是什么（定义、例子）
├─ 分支 2：特点（优缺点）
├─ 分支 3：操作（增删改查）
├─ 分支 4：复杂度（时间分析）
└─ 分支 5：应用（实际场景）
```
