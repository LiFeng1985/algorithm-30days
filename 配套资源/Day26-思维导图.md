## 分支 1：0-1 背包问题

```
0-1 背包
    │
    ├─ 问题描述
    │   ├─ n 个物品
    │   ├─ 每个有重量和价值
    │   ├─ 背包容量限制
    │   └─ 最大化总价值
    │
    ├─ 状态定义
    │   └─ dp[i][j]
    │       └─ 前 i 个物品放入容量 j 的背包
    │           的最大价值
    │
    ├─ 状态转移
    │   ├─ 不放：dp[i-1][j]
    │   └─ 放：dp[i-1][j-w[i]] + v[i]
    │       取 max
    │
    ├─ 空间优化
    │   └─ 一维数组
    │       从后往前遍历
    │
    └─ 实际应用
        ├─ 投资组合
        └─ 资源分配
```

---

## 分支 2：最长公共子序列（LCS）

```
LCS
    │
    ├─ 问题描述
    │   ├─ 两个字符串
    │   ├─ 找最长的公共子序列
    │   └─ 子序列不要求连续
    │
    ├─ 状态定义
    │   └─ dp[i][j]
    │       └─ text1 前 i 个和 text2 前 j 个
    │           的 LCS 长度
    │
    ├─ 状态转移
    │   ├─ 字符相同：dp[i-1][j-1] + 1
    │   └─ 字符不同：max(dp[i-1][j], dp[i][j-1])
    │
    ├─ 回溯找 LCS
    │   └─ 从 dp[m][n] 倒推
    │
    └─ 实际应用
        ├─ DNA 比对
        └─ Git diff
```

## 分支 3：编辑距离

```
编辑距离
    │
    ├─ 问题描述
    │   ├─ 两个单词
    │   ├─ 计算最少操作数
    │   └─ 使 word1 变成 word2
    │
    ├─ 三种操作
    │   ├─ 插入一个字符
    │   ├─ 删除一个字符
    │   └─ 替换一个字符
    │
    ├─ 状态定义
    │   └─ dp[i][j]
    │       └─ word1 前 i 个转 word2 前 j 个
    │           的最少操作数
    │
    ├─ 状态转移
    │   ├─ 字符相同：dp[i-1][j-1]
    │   └─ 字符不同：min(插入，删除，替换) + 1
    │
    └─ 实际应用
        ├─ 拼写检查
        └─ 语音识别
```

## 分支 4：对比总结

```
对比
    │
    ├─ 相同点
    │   ├─ 都是二维 DP
    │   ├─ 时间复杂度 O(m×n)
    │   ├─ 空间复杂度 O(m×n)
    │   └─ 都有最优子结构
    │
    ├─ 不同点
    │   ├─ 问题类型
    │   │   ├─ 背包：最优化
    │   │   ├─ LCS: 计数
    │   │   └─ 编辑距离：最优化
    │   ├─ 状态定义不同
    │   └─ 转移方程不同
    │
    └─ 解题技巧
        ├─ 先理解问题
        ├─ 定义合适的状态
        ├─ 推导转移方程
        └─ 考虑空间优化
```

## 🔑 核心要点

### 三个关键词
```
1️⃣ 状态定义
   明确 dp[i][j] 的含义

2️⃣ 状态转移
   从小问题推大问题

3️⃣ 边界条件
   初始化好 dp[0][...] 和 dp[...][0]
```

### 一个中心思想
```
💡 分而治之
将复杂问题分解为重叠的子问题
用表格存储子问题的解
```

### 重要技巧
```
✓ 先写暴力递归
✓ 再加记忆化
✓ 最后改递推
✓ 能优化就优化空间
```

## 💡 学习建议

### 理解重点
```
1. 掌握三个问题的状态定义
2. 理解状态转移方程的推导
3. 明白为什么要这样定义
4. 学会空间优化的技巧
```

### 练习方向
```
✓ 能手写 0-1 背包的两种实现
✓ 能手写 LCS 并回溯路径
✓ 能手写编辑距离
✓ 能分析时间和空间复杂度
```

### 常见误区
```
✗ 状态定义不清晰
✗ 忘记初始化边界
✗ 遍历顺序错误
✗ 空间优化时方向错
```

## 📊 DP 问题解题模板

```
解决 DP 问题的通用步骤：

1. 理解问题
   - 是什么类型？（最优化/计数/存在性）
   - 有什么限制条件？

2. 定义状态
   - dp[i][j] 表示什么？
   - 最终答案是什么？

3. 找状态转移方程
   - 如何从已知推未知？
   - 有几种情况需要考虑？

4. 确定初始条件
   - dp[0][...] = ?
   - dp[...][0] = ?

5. 确定计算顺序
   - 从小到大？
   - 确保依赖的状态已计算

6. 实现代码
   - 嵌套循环
   - 注意边界

7. 空间优化（可选）
   - 能否降维？
   - 滚动数组？
```
