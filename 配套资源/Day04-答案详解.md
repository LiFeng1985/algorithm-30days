## 一、基础概念题答案

### 题目 1：什么是栈？（10 分）

**参考答案：**
```
栈是一种后进先出（LIFO）的数据结构，
就像一摞盘子，只能从最上面拿或放。
所有操作都在栈顶进行。
```

### 题目 2：LIFO 的含义（10 分）

**参考答案：**

**含义：** Last In First Out，后进先出
```
最后放进去的元素，第一个被拿出来。
就像往瓶子里装弹珠，最后装的在最上面，
倒出来的时候最先出来。
```

**生活例子：**
- 羽毛球筒：最后放的球最先拿出来
- 浏览器后退：最后访问的页面最先回去
- 洗碗：最后洗的碗在最上面，最先用

**评分要点：**
- 解释清楚 LIFO（5 分）
- 例子恰当（5 分）

## 二、代码实践题答案

### 题目 4：实现栈类（25 分）

**参考答案：**

```javascript
class Stack {
    constructor() {
        this.items = [];  // 用数组存储
    }
    
    push(element) {
        this.items.push(element);
    }
    
    pop() {
        if (this.isEmpty()) {
            return null;  // 栈空返回 null
        }
        return this.items.pop();
    }
    
    peek() {
        if (this.isEmpty()) {
            return null;
        }
        return this.items[this.items.length - 1];
    }
    
    isEmpty() {
        return this.items.length === 0;
    }
    
    size() {
        return this.items.length;
    }
}
```

**评分要点：**
- 构造函数正确（3 分）
- push 正确（4 分）
- pop 正确（6 分，含边界检查 2 分）
- peek 正确（6 分，含边界检查 2 分）
- isEmpty 正确（3 分）
- size 正确（3 分）

### 题目 5：括号匹配（20 分）

**参考答案：**

```javascript
function isValid(s) {
    const stack = new Stack();
    const map = {
        ')': '(',
        ']': '[',
        '}': '{'
    };
    
    for (let char of s) {
        if (char === '(' || char === '[' || char === '{') {
            stack.push(char);  // 左括号入栈
        } else {
            const top = stack.pop();
            if (top !== map[char]) {
                return false;  // 不匹配
            }
        }
    }
    
    return stack.isEmpty();  // 栈空说明全部匹配
}
```

**评分要点：**
- 使用栈（5 分）
- 左括号处理正确（5 分）
- 右括号匹配逻辑正确（5 分）
- 最后检查栈空（5 分）

### 题目 6：逆波兰表达式求值（15 分）

**参考答案：**

```javascript
function evalRPN(tokens) {
    const stack = new Stack();
    
    for (let token of tokens) {
        if (token === '+' || token === '-' || 
            token === '*' || token === '/') {
            const b = stack.pop();
            const a = stack.pop();
            
            switch(token) {
                case '+': stack.push(a + b); break;
                case '-': stack.push(a - b); break;
                case '*': stack.push(a * b); break;
                case '/': stack.push(parseInt(a / b)); break;
            }
        } else {
            stack.push(parseInt(token));
        }
    }
    
    return stack.pop();
}
```

**评分要点：**
- 使用栈存储数字（3 分）
- 遇到运算符弹出两个数（4 分）
- 计算顺序正确（a 和 b 的顺序）（4 分）
- 除法取整正确（2 分）
- 返回最终结果（2 分）

### 题目 8：函数调用栈（10 分）

**参考答案：**

```
函数调用要用栈是因为：

1. 需要保存返回地址
   函数执行完要知道回到哪里

2. 需要保存局部变量
   不同函数的变量不能互相干扰

3. 支持递归调用
   同一个函数可以嵌套调用多次

4. 符合 LIFO 原则
   最后调用的函数最先返回
```

**评分要点：**
- 提到保存返回地址（3 分）
- 提到保存局部变量（3 分）
- 提到支持递归（2 分）
- 提到 LIFO 特性（2 分）
