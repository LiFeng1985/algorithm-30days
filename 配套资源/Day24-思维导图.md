## 分支 1：BFS（广度优先搜索）

```
BFS
    │
    ├─ 核心思想
    │   └─ 层层推进
    │       ├─ 先近后远
    │       └─ 水波纹扩散
    │
    ├─ 实现方法
    │   ├─ 队列（FIFO）
    │   ├─ visited 集合
    │   └─ 按层遍历
    │
    ├─ 复杂度
    │   ├─ 时间：O(V+E)
    │   └─ 空间：O(V)
    │
    └─ 特点
        ├─ 能找到最短路径
        ├─ 不会陷入死胡同
        └─ 空间消耗可能大
```

---

## 分支 2：DFS（深度优先搜索）

```
DFS
    │
    ├─ 核心思想
    │   └─ 一条路走到底
    │       ├─ 不撞南墙不回头
    │       └─ 回溯
    │
    ├─ 实现方法
    │   ├─ 栈（LIFO）
    │   ├─ 递归
    │   └─ visited 集合
    │
    ├─ 复杂度
    │   ├─ 时间：O(V+E)
    │   └─ 空间：O(V)
    │
    └─ 特点
        ├─ 能枚举所有可能
        ├─ 实现简单
        └─ 可能栈溢出
```

---

## 分支 3：对比分析

```
对比
    │
    ├─ 相同点
    │   ├─ 都是图遍历算法
    │   ├─ 时间复杂度 O(V+E)
    │   └─ 都需要 visited
    │
    ├─ 不同点
    │   ├─ 遍历策略
    │   │   ├─ BFS: 层层推进
    │   │   └─ DFS: 深入探索
    │   ├─ 数据结构
    │   │   ├─ BFS: 队列
    │   │   └─ DFS: 栈/递归
    │   └─ 空间特性
    │       ├─ BFS: 存储整层
    │       └─ DFS: 存储路径
    │
    └─ 选择原则
        ├─ 找最短路径 → BFS
        ├─ 枚举所有解 → DFS
        ├─ 图很宽 → DFS
        └─ 图很深 → BFS
```

## 🔑 核心要点

### 三个关键词
```
1️⃣ 层次
   BFS 按层遍历

2️⃣ 深入
   DFS 一路到底

3️⃣ 回溯
   DFS 的核心技巧
```

### 一个中心思想
```
💡 遍历策略决定适用场景
根据问题特点选择合适的算法
```

### 重要技巧
```
✓ BFS 用队列，DFS 用栈/递归
✓ 都要用 visited 避免重复访问
✓ BFS 适合最短路径
✓ DFS 适合枚举所有可能
```

---

## 💡 学习建议

### 理解重点
```
1. 掌握两种算法的实现
2. 理解遍历策略的区别
3. 明白各自的适用场景
4. 知道如何选择算法
```

### 练习方向
```
✓ 能手写 BFS 实现
✓ 能手写 DFS 实现（递归 + 迭代）
✓ 能用 BFS 解决最短路径
✓ 能用 DFS 解决回溯问题
```

### 常见误区
```
✗ 混淆队列和栈的使用
✗ 忘记标记 visited
✗ 认为 BFS 总是比 DFS 好
✗ 不理解为什么复杂度相同
```

## 📊 算法选择决策树

```
需要解决的问题是什么？
│
├─ 找最短路径（无权图）？
│   └─ 是 → 用 BFS
│
├─ 枚举所有可能的解？
│   └─ 是 → 用 DFS
│
├─ 图的结构很宽？
│   └─ 是 → 用 DFS（省空间）
│
├─ 图的结构很深？
│   └─ 是 → 用 BFS（避免栈溢出）
│
└─ 不确定？
    ├─ 需要最优解 → BFS
    └─ 需要所有解 → DFS
```
