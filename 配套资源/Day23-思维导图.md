## 🔑 核心要点

### 三个关键词
```
1️⃣ 映射
   键→索引的快速转换

2️⃣ 冲突
   不同键映射到同索引

3️⃣ O(1)
   平均常数时间复杂度
```

### 一个中心思想
```
💡 空间换时间
用额外的存储空间换取查找速度
```

### 重要技巧
```
✓ 使用质数乘法使分布均匀
✓ 链地址法简单易实现
✓ 负载因子控制性能
✓ 适时扩容保持效率
```
