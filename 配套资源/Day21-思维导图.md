## 🔑 核心要点

### 三个层次
```
1️⃣ 简单排序（O(n²)）
   冒泡、选择、插入
   适合小数据

2️⃣ 高效排序（O(n log n)）
   归并、快速、堆排序
   适合大数据

3️⃣ 特殊排序
   计数、桶、基数排序
   适合特定场景
```

### 一个中心思想
```
💡 没有最好的排序，只有最合适的排序
根据具体情况选择最适合的算法
```

### 重要技巧
```
✓ 小数据用简单排序
✓ 基本有序用插入排序
✓ 大规模用快速排序
✓ 要稳定用归并排序
✓ 内存紧用堆排序
```
