## 🔑 核心要点

### 三个关键词
```
1️⃣ 分区
   小的在左，大的在右

2️⃣ 递归
   对两边重复操作

3️⃣ 原地
   不需要额外空间
```

### 一个中心思想
```
💡 分而治之
通过分区将大问题分解为小问题
```

### 重要技巧
```
✓ pivot 选择影响性能
✓ Lomuto 分区简单易实现
✓ 随机化避免最坏情况
```

## 💡 学习建议

### 理解重点
```
1. 掌握 partition 的实现
2. 理解分治法的应用
3. 明白为什么平均 O(n log n)
4. 知道如何避免最坏情况
```

### 练习方向
```
✓ 手写 partition 函数
✓ 手写 quickSort 递归
✓ 手动模拟分区过程
✓ 分析最好/最坏情况
```

### 常见误区
```
✗ pivot 选择不当导致 O(n²)
✗ 忘记基准情况导致死循环
✗ 分区边界处理错误
✗ 误以为是稳定排序
```
