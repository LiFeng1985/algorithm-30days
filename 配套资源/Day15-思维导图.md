## 分支 1：基本原理

```
原理
    │
    ├─ 基本思想
    │   ├─ 相邻比较
    │   └─ 交换位置
    │
    ├─ 名称由来
    │   └─ 大的元素像气泡上浮
    │
    └─ 执行过程
        ├─ 每轮将最大元素移到末尾
        └─ 轮数：n-1 轮
```

---

## 分支 2：代码实现

```
实现
    │
    ├─ 基础版本
    │   ├─ 两层循环
    │   ├─ 外层控制轮数
    │   └─ 内层进行比较交换
    │
    ├─ 关键代码
    │   ├─ if (arr[j] > arr[j+1])
    │   └─ 交换操作
    │
    └─ 注意事项
        ├─ 内层边界：j < n-i-1
        └─ 返回排序后的数组
```

---

## 分支 3：优化方法

```
优化
    │
    ├─ 标志位优化
    │   ├─ 记录是否发生交换
    │   └─ 无交换则提前结束
    │
    ├─ 性能提升
    │   ├─ 最好情况：O(n²) → O(n)
    │   └─ 平均情况也有提升
    │
    └─ 其他优化思路
        ├─ 双向冒泡（鸡尾酒排序）
        └─ 记录最后交换位置
```

---

## 分支 4：复杂度与特性

```
复杂度与特性
    │
    ├─ 时间复杂度
    │   ├─ 最好：O(n) - 已有序
    │   ├─ 平均：O(n²)
    │   └─ 最坏：O(n²) - 完全逆序
    │
    ├─ 空间复杂度
    │   └─ O(1) - 原地排序
    │
    ├─ 稳定性
    │   └─ 稳定排序
    │
    └─ 适用场景
        ├─ 数据量小
        ├─ 基本有序
        └─ 教学演示
```

---

## 🔑 核心要点

### 三个关键词
```
1️⃣ 相邻比较
   每次比较相邻的两个元素

2️⃣ 交换位置
   顺序错误就交换

3️⃣ 提前结束
   无交换说明已有序
```

### 一个中心思想
```
💡 逐步逼近
每一轮都让一个元素到达正确位置
```
