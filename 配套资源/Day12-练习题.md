## 💡 提示与启发

### 解题思路提示

**题目 4：**
- 用数组存储顶点列表
- 用 Map 存储邻接表
- 注意有向图和无向图的区别

**题目 5：**
- 使用 BFS 或 DFS 遍历
- 用 Set 记录已访问的顶点
- 找到目标就返回 true

**题目 6：**
- 无向图：度 = 邻居数量
- 有向图：入度 = 指向我的边数，出度 = 我指出的边数

### 常见错误提醒

❌ 忘记处理重复顶点 → ✅ 添加前检查是否已存在
❌ 有向图双向添加 → ✅ 有向图只添加单向边
❌ 删除顶点未清理边 → ✅ 删除顶点时要删除相关边
❌ 忘记初始化 → ✅ 新顶点要初始化空邻居列表
