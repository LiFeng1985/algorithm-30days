### 题目 3：二叉树的特点（10 分）

**参考答案：**

**特点 1：每个节点最多 2 个子节点**（3 分）
```
不能有 3 个或更多子节点
可以有 0 个、1 个或 2 个子节点
```

**特点 2：子树有左右之分**（3 分）
```
左子树和右子树顺序不能颠倒
即使只有一个子节点，也要区分左右
```

**特点 3：递归定义**（2 分）
```
每个子树本身也是二叉树
可以用递归的方法处理
```

**其他可接受答案：**
- 第 i 层最多有 2^(i-1) 个节点
- 深度为 k 的树最多有 2^k - 1 个节点

**评分要点：**
- 每个特点 3 分左右

### 题目 5：实现三种遍历（30 分）

**参考答案：**

```javascript
/**
 * 前序遍历：根→左→右
 */
function preorder(root) {
    if (root === null) return;
    
    console.log(root.val);      // 访问根节点
    preorder(root.left);        // 遍历左子树
    preorder(root.right);       // 遍历右子树
}

/**
 * 中序遍历：左→根→右
 */
function inorder(root) {
    if (root === null) return;
    
    inorder(root.left);         // 遍历左子树
    console.log(root.val);      // 访问根节点
    inorder(root.right);        // 遍历右子树
}

/**
 * 后序遍历：左→右→根
 */
function postorder(root) {
    if (root === null) return;
    
    postorder(root.left);       // 遍历左子树
    postorder(root.right);      // 遍历右子树
    console.log(root.val);      // 访问根节点
}
```

**执行过程演示：**
```
    1
   / \
  2   3

前序遍历：
preorder(1)
→ 输出 1
→ preorder(2) → 输出 2
→ preorder(3) → 输出 3
结果：1 2 3 ✓

中序遍历：
inorder(1)
→ inorder(2) → 输出 2
→ 输出 1
→ inorder(3) → 输出 3
结果：2 1 3 ✓

后序遍历：
postorder(1)
→ postorder(2) → 输出 2
→ postorder(3) → 输出 3
→ 输出 1
结果：2 3 1 ✓
```

**评分要点：**
- 前序遍历正确（10 分）
- 中序遍历正确（10 分）
- 后序遍历正确（10 分）
- 每种遍历：顺序正确 6 分，递归调用正确 4 分

**常见错误：**
❌ 三种遍历顺序搞混 → ✅ 记住口诀：前序=根左右，中序=左根右，后序=左右根
❌ 忘记判空 → ✅ 第一句必须是 `if (root === null) return;`

### 题目 6：计算树的最大深度（15 分）

**参考答案：**

```javascript
function maxDepth(root) {
    // 基准情况：空树深度为 0
    if (root === null) {
        return 0;
    }
    
    // 递归计算左右子树的深度
    const leftDepth = maxDepth(root.left);
    const rightDepth = maxDepth(root.right);
    
    // 返回较深的子树深度 + 1（当前节点）
    return Math.max(leftDepth, rightDepth) + 1;
}

// 测试
//   1
//  / \
// 2   3
//    /
//   4
const root = new TreeNode(1);
root.left = new TreeNode(2);
root.right = new TreeNode(3);
root.right.left = new TreeNode(4);

console.log(maxDepth(root));  // 3
```

**执行过程：**
```
maxDepth(节点 1)
= max(maxDepth(2), maxDepth(3)) + 1
= max(1, max(maxDepth(3 的左), maxDepth(3 的右))) + 1
= max(1, max(maxDepth(4), 0)) + 1
= max(1, max(1, 0)) + 1
= max(1, 1) + 1
= 1 + 1
= 2... 等等，让我重新算

正确过程：
maxDepth(4) = 1（叶节点）
maxDepth(3) = max(maxDepth(4), 0) + 1 = max(1, 0) + 1 = 2
maxDepth(2) = 1（叶节点）
maxDepth(1) = max(1, 2) + 1 = 3 ✓
```

**评分要点：**
- 基准情况正确（4 分）
- 递归调用正确（6 分）
- 返回结果正确（含 +1）（3 分）
- 能通过测试（2 分）

### 题目 8：满二叉树 vs 完全二叉树（10 分）

**参考答案：**

**满二叉树：**（4 分）
```
每一层的节点数都达到最大值
所有叶节点都在同一层
例如：
    1
   / \
  2   3
 / \ / \
4  5 6  7
```

**完全二叉树：**（4 分）
```
除了最后一层，其他层都是满的
最后一层的节点都靠左排列
可以有空缺，但必须从右边开始缺
例如：
    1
   / \
  2   3
 / \ /
4  5 6
```

**区别：**（2 分）
```
满二叉树要求更严格，必须每层都满
完全二叉树允许最后一层不满，但要靠左
满二叉树一定是完全二叉树
完全二叉树不一定是满二叉树
```

**评分要点：**
- 满二叉树定义正确（4 分）
- 完全二叉树定义正确（4 分）
- 说清楚区别（2 分）

## 💡 错题分析

### 常见错误类型

**1. 遍历顺序混淆**
```
错误：前序遍历写成左根右
纠正：前序=根左右，记住"前"就是根在前
练习：多画几次遍历过程
```

**2. 递归忘记判空**
```
错误：直接访问 root.left 导致报错
纠正：第一件事检查 root === null
记忆：先判空，再操作
```

**3. 深度计算忘 +1**
```
错误：return Math.max(left, right)
纠正：当前节点也算一层，要 +1
理解：高度 = 子树最大高度 + 自己这层
```

### 改进建议

```
□ 多画图理解树的结构
□ 手写三种遍历各 5 遍
□ 理解递归的执行过程
□ 向他人解释树的概念
□ 不要怕错，从错误中学习
```
