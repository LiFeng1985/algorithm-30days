---

## ✅ 学习目标检查

### 知识理解
- [ ] 理解什么是算法（能用自己的话说）
- [ ] 明白为什么算法很重要
- [ ] 知道算法在日常生活中的应用
- [ ] 理解"程序 = 数据结构 + 算法"的含义

### 核心概念
- [ ] 能解释"算法是解决问题的步骤"
- [ ] 理解好算法的标准（快、省、准）
- [ ] 知道算法效率的重要性
- [ ] 了解时间复杂度的概念（初步）

### 思维能力
- [ ] 能识别生活中的算法例子
- [ ] 会比较不同解决方法的优劣
- [ ] 开始用算法思维思考问题
- [ ] 对后续学习产生兴趣

- [ ] 算法只存在于计算机程序中
- [ ] 菜谱也是一种算法
- [ ] 算法越复杂越好
- [ ] 同一个问题只能有一种算法解决
