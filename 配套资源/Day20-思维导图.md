## 🔑 核心要点

### 三个关键词
```
1️⃣ 建堆
   从最后一个非叶节点开始

2️⃣ 交换
   堆顶与末尾交换

3️⃣ 调整
   heapify 维护堆性质
```

### 一个中心思想
```
💡 反复提取最大值
利用大顶堆每次获取最大值
```

### 重要技巧
```
✓ 最后一个非叶节点 = Math.floor(n/2) - 1
✓ 左子节点 = 2i + 1
✓ 右子节点 = 2i + 2
✓ 堆大小逐次减 1
```
