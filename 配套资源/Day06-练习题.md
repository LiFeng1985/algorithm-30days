## 💡 提示与启发

### 解题思路提示

**题目 4：**
- 基准情况：n = 0 或 n = 1 时返回 1
- 递归：n! = n × (n-1)!

**题目 5：**
- 基准情况：F(1) = 1, F(2) = 1
- 递归：F(n) = F(n-1) + F(n-2)

**题目 6：**
- 基准情况：空数组返回 0
- 递归：sum(arr) = arr[0] + sum(rest)

### 常见错误提醒

❌ 忘记基准情况 → ✅ 先写基准情况
❌ 递归没有进展 → ✅ 确保向基准情况靠近
❌ 栈溢出 → ✅ 检查递归深度
