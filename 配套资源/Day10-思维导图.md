## 分支 1：堆的定义

```
堆定义
    │
    ├─ 结构特点
    │   ├─ 完全二叉树
    │   └─ 数组存储
    │
    ├─ 类型
    │   ├─ 大顶堆
    │   │   └─ 父节点 ≥ 子节点
    │   └─ 小顶堆
    │       └─ 父节点 ≤ 子节点
    │
    └─ 索引关系
        ├─ parent = (i-1)/2
        ├─ left = 2i+1
        └─ right = 2i+2
```

---

## 分支 2：基本操作

```
堆操作
    │
    ├─ 插入 Insert
    │   ├─ 放在末尾
    │   └─ 上滤调整
    │
    ├─ 删除 Delete
    │   ├─ 替换为末尾
    │   └─ 下滤调整
    │
    ├─ 查找 Max/Min
    │   └─ O(1) 直接访问根节点
    │
    └─ 建堆 Heapify
        └─ O(n) 从最后一个非叶节点开始
```

## 分支 3：优先队列

```
优先队列
    │
    ├─ 定义
    │   └─ 优先级高的先出队
    │
    ├─ 实现方式
    │   └─ 用堆实现（最高效）
    │
    ├─ 操作
    │   ├─ enqueue 入队
    │   ├─ dequeue 出队
    │   ├─ peek 查看
    │   └─ isEmpty 判空
    │
    └─ 应用场景
        ├─ 任务调度
        └─ Top K 问题
```

---

## 分支 4：应用与复杂度

```
应用与复杂度
    │
    ├─ 时间复杂度
    │   ├─ 插入：O(log n)
    │   ├─ 删除：O(log n)
    │   ├─ 查找：O(1)
    │   └─ 建堆：O(n)
    │
    ├─ 空间复杂度
    │   └─ O(n)
    │
    └─ 实际应用
        ├─ 堆排序
        ├─ Top K 问题
        ├─ 中位数查找
        └─ 图算法（Dijkstra）
```

---

## 🔑 核心要点

### 三个关键词
```
1️⃣ 完全二叉树
   结构紧凑，数组存储

2️⃣ 上滤下滤
   插入时上滤，删除时下滤

3️⃣ 优先队列
   堆的最佳应用场景
```

### 一个中心思想
```
💡 局部有序带来全局高效
根节点总是最大（或最小）
```
