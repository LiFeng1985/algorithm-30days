## 🤔 一、先搞懂一个问题

### **为什么需要红黑树？**

```javascript
// 普通的树可能长这样（歪的）：
    1
     \
      2
       \
        3
         \
          4

查找 4 要走 4 步！慢！😫


// 红黑树保证平衡：
    2
   / \
  1   3
       \
        4

查找 4 只要走 2 步！快！😊

就这么简单！
```

---

## 🎨 二、红黑树就 3 个核心

### **1. 节点有颜色**

```javascript
每个节点要么红色 🔴，要么黑色 ⚫

就这个！没啥复杂的！
```

---

### **2. 旋转保持平衡**

```javascript
右边重了 → 左旋（把右边的提起来）
左边重了 → 右旋（把左边的提起来）

就像玩跷跷板：
哪边重，就把哪边提起来！
```

---

### **3. 变色调整**

```javascript
红色太多了 → 变几个成黑色
黑色太多了 → 变几个成红色

就像换衣服一样简单！
```

## 💻 三、代码实现（简化版）

### **只需要记住这些：**

```javascript
class Node {
    constructor(data) {
        this.data = data;
        this.color = 'RED';  // 新节点都是红色
        this.left = null;
        this.right = null;
    }
}

class RedBlackTree {
    constructor() {
        this.root = null;
    }
    
    insert(data) {
        this.root = this._insert(this.root, data);
        this.root.color = 'BLACK';  // 根是黑色
    }
    
    _insert(node, data) {
        if (!node) return new Node(data);
        
        if (data < node.data) {
            node.left = this._insert(node.left, data);
        } else {
            node.right = this._insert(node.right, data);
        }
        
        return this._fix(node);  // 修复
    }
    
    _fix(node) {
        // 右边红左边黑 → 左旋
        if (this._isRed(node.right) && !this._isRed(node.left)) {
            node = this._rotateLeft(node);
        }
        
        // 左边红左左红 → 右旋
        if (this._isRed(node.left) && this._isRed(node.left.left)) {
            node = this._rotateRight(node);
        }
        
        // 左右都红 → 变色
        if (this._isRed(node.left) && this._isRed(node.right)) {
            this._flipColors(node);
        }
        
        return node;
    }
    
    _rotateLeft(node) {
        const newRoot = node.right;
        node.right = newRoot.left;
        newRoot.left = node;
        newRoot.color = node.color;
        node.color = 'RED';
        return newRoot;
    }
    
    _rotateRight(node) {
        const newRoot = node.left;
        node.left = newRoot.right;
        newRoot.right = node;
        newRoot.color = node.color;
        node.color = 'RED';
        return newRoot;
    }
    
    _flipColors(node) {
        node.color = node.color === 'RED' ? 'BLACK' : 'RED';
        node.left.color = node.left.color === 'RED' ? 'BLACK' : 'RED';
        node.right.color = node.right.color === 'RED' ? 'BLACK' : 'RED';
    }
    
    _isRed(node) {
        return node && node.color === 'RED';
    }
}

// 使用
const tree = new RedBlackTree();
tree.insert(40);
tree.insert(20);
tree.insert(60);
tree.insert(10);
tree.insert(30);
console.log('完成！');
```
