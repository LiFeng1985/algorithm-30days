## 📊 二、完整正确代码

```javascript
class PriorityQueue {
    constructor() {
        this.heap = [];
    }
    
    enqueue(value) {
        this.insert(value);
    }
    
    dequeue() {
        return this.extractMax();
    }
    
    insert(key) {
        this.heap.push(key);
        this.shiftUp(this.heap.length - 1);
    }
    
    extractMax() {
        if (this.heap.length === 0) return null;
        
        const max = this.heap[0];
        this.heap[0] = this.heap.pop();
        
        if (this.heap.length > 0) {
            this.shiftDown(0);
        }
        
        return max;
    }
    
    shiftUp(i) {
        while (i > 0 && this.heap[Math.floor((i - 1) / 2)] < this.heap[i]) {
            const parent = Math.floor((i - 1) / 2);
            [this.heap[i], this.heap[parent]] = [this.heap[parent], this.heap[i]];
            i = parent;
        }
    }
    
    shiftDown(i) {
        const left = 2 * i + 1;
        const right = 2 * i + 2;
        let largest = i;
        const heapSize = this.heap.length;
        
        if (left < heapSize && this.heap[left] > this.heap[largest]) {
            largest = left;
        }
        
        if (right < heapSize && this.heap[right] > this.heap[largest]) {
            largest = right;
        }
        
        if (largest !== i) {
            [this.heap[i], this.heap[largest]] = [this.heap[largest], this.heap[i]];
            this.shiftDown(largest);
        }
    }
    
    peek() {
        return this.heap.length === 0 ? null : this.heap[0];
    }
}
```

## 🎨 三、核心概念图解

### **堆的结构：**

```
最大堆性质：
1. 完全二叉树
2. 每个节点的值 >= 其子节点的值

数组表示：[100, 90, 80, 70, 60, 50, 40]

对应的树结构：
          100 (索引 0)
         /   \
       90     80  (索引 1, 2)
      /  \   /  \
    70   60 50  40 (索引 3,4,5,6)

索引关系：
- 父节点：parent = floor((i - 1) / 2)
- 左孩子：left = 2 * i + 1
- 右孩子：right = 2 * i + 2
```

## 💡 四、insert + shiftUp 详解

### **插入过程：**

```javascript
insert(key) {
    this.heap.push(key);           // 1. 添加到末尾
    this.shiftUp(this.heap.length - 1);  // 2. 上浮到正确位置
}
```

### **示例：插入 95**

```
初始堆：[100, 90, 80, 70, 60, 50, 40]

步骤 1：push(95)
[100, 90, 80, 70, 60, 50, 40, 95]
                  ↑
                新元素

对应的树：
          100
         /   \
       90     80
      /  \   /  \
    70   60 50  40
   /
 95  ← 新节点

步骤 2：shiftUp(95)
比较 95 和父节点 70：95 > 70 → 交换
          100
         /   \
       90     80
      /  \   /  \
    95   60 50  40
   /
 70

继续比较 95 和父节点 90：95 > 90 → 交换
          100
         /   \
       95     80
      /  \   /  \
    90   60 50  40
   /
 70

继续比较 95 和父节点 100：95 < 100 → 停止！✅

最终堆：[100, 95, 80, 90, 60, 50, 40, 70]
```

### **shiftUp 代码逐行解析：**

```javascript
shiftUp(i) {
    // 当 i > 0（不是根）且父节点比当前节点小
    while (i > 0 && this.heap[Math.floor((i - 1) / 2)] < this.heap[i]) {
        
        const parent = Math.floor((i - 1) / 2);  // 计算父节点索引
        
        // 交换当前节点和父节点
        [this.heap[i], this.heap[parent]] = [this.heap[parent], this.heap[i]];
        
        i = parent;  // 更新当前位置，继续上浮
    }
}
```

**关键点：**
- ✅ 循环条件：`i > 0`（没到根）且 `父节点 < 当前节点`
- ✅ 每次交换后，`i` 上移到父节点位置
- ✅ 时间复杂度：O(log n) - 最多上升树的高度

## 🔍 五、extractMax + shiftDown 详解

### **删除堆顶过程：**

```javascript
extractMax() {
    if (this.heap.length === 0) return null;
    
    const max = this.heap[0];      // 1. 保存最大值
    this.heap[0] = this.heap.pop(); // 2. 最后一个元素移到根
    
    if (this.heap.length > 0) {
        this.shiftDown(0);         // 3. 下沉修复堆性质
    }
    
    return max;
}
```

### **示例：删除最大值**

```
初始堆：[100, 95, 80, 90, 60, 50, 40, 70]

对应的树：
          100
         /   \
       95     80
      /  \   /  \
    90   60 50  40
   /
 70

步骤 1：保存 max = 100
步骤 2：把最后一个元素 70 移到根，并删除末尾
          70
         /   \
       95     80
      /  \   /  \
    90   60 50  40

步骤 3：shiftDown(70)
比较 70 的子节点：95 和 80
95 > 80，所以和 95 交换
          95
         /   \
       70     80
      /  \   /  \
    90   60 50  40

继续：70 的子节点是 90 和 60
90 > 60，且 90 > 70 → 交换
          95
         /   \
       90     80
      /  \   /  \
    70   60 50  40

继续：70 的子节点... 没有子节点了 → 停止！✅

最终堆：[95, 90, 80, 70, 60, 50, 40]
返回：100
```

### **shiftDown 代码逐行解析：**

```javascript
shiftDown(i) {
    const left = 2 * i + 1;       // 左孩子索引
    const right = 2 * i + 2;      // 右孩子索引
    let largest = i;              // 假设当前节点最大
    const heapSize = this.heap.length;
    
    // 如果左孩子存在且更大
    if (left < heapSize && this.heap[left] > this.heap[largest]) {
        largest = left;
    }
    
    // 如果右孩子存在且更大
    if (right < heapSize && this.heap[right] > this.heap[largest]) {
        largest = right;
    }
    
    // 如果最大值不是当前节点
    if (largest !== i) {
        // 交换当前节点和较大的子节点
        [this.heap[i], this.heap[largest]] = [this.heap[largest], this.heap[i]];
        
        // 递归处理被交换的子树
        this.shiftDown(largest);
    }
}
```

**关键点：**
- ✅ 找出当前节点、左孩子、右孩子中的最大值
- ✅ 如果最大值不是当前节点，就交换
- ✅ 递归处理被交换的子树
- ✅ 时间复杂度：O(log n) - 最多下沉树的高度

## 📈 七、时间复杂度分析

| 操作 | 时间复杂度 | 空间复杂度 | 说明 |
|------|-----------|-----------|------|
| **insert** | O(log n) | O(1) | 上浮操作 |
| **extractMax** | O(log n) | O(1) | 下沉操作 |
| **peek** | O(1) | O(1) | 直接访问堆顶 |
| **enqueue** | O(log n) | O(1) | 调用 insert |
| **dequeue** | O(log n) | O(1) | 调用 extractMax |

**为什么是 O(log n)？**

```
堆是完全二叉树，高度 h = log₂(n)

insert:
- 最坏情况：从叶子一直上浮到根
- 移动次数 = 树的高度 = O(log n)

extractMax:
- 最坏情况：从根一直下沉到叶子
- 移动次数 = 树的高度 = O(log n)
```

## 🎯 八、常见错误总结

### **错误 1：变量名混淆**

```javascript
// ❌ 错误
if (left < heapSize && arr[left] > arr[largest]) {
    [arr[i], arr[largest]] = [arr[largest], arr[i]];
}

// ✅ 正确
if (left < heapSize && this.heap[left] > this.heap[largest]) {
    [this.heap[i], this.heap[largest]] = [this.heap[largest], this.heap[i]];
}

原因：类的属性是 this.heap，不是 arr！
```

---

### **错误 2：缺少 this**

```javascript
// ❌ 错误
extractMax() {
    if (this.heap.length > 0) {
        shiftDown(0);  // ReferenceError: shiftDown is not defined
    }
}

// ✅ 正确
extractMax() {
    if (this.heap.length > 0) {
        this.shiftDown(0);  // 调用类的方法
    }
}
```

---

### **错误 3：参数不匹配**

```javascript
// ❌ 错误
shiftDown(i) { ... }  // 定义时只有 1 个参数
this.shiftDown(largest, heapSize);  // 调用时传 2 个参数

// ✅ 正确
shiftDown(i) { ... }  // 定义时 1 个参数
this.shiftDown(largest);  // 调用时也传 1 个参数
```

### **错误 4：忘记更新 i**

```javascript

    }

## ✅ 十、总结

### **关键要点：**

```
1. ⭐ 最大堆性质：父节点 >= 子节点

2. ⭐ 两个核心操作：
   - shiftUp：插入时上浮
   - shiftDown：删除时下沉

3. ⭐ 索引关系：

   - 插入：O(log n)
   - 删除：O(log n)

   - 使用 this.heap，不是 arr
   - 调用方法要加 this
   - 参数要匹配
   - 记得更新位置索引
