## 🎯 二、快速排序核心要点（5 分钟）

### **一句话总结：**

> **选个基准，分成两堆，小的左大的右，递归继续！**

---

### **三个关键概念：**

#### **1. 分区（Partition）- 最核心的部分！**

```javascript
function partition(arr, low, high) {
    const pivot = arr[high];    // 选基准（最后一个元素）
    let i = low - 1;            // i 指向小于区的末尾
    
    for (let j = low; j < high; j++) {
        if (arr[j] < pivot) {   // 发现比基准小的
            i++;                // 扩展小于区
            [arr[i], arr[j]] = [arr[j], arr[i]];  // 交换
        }
    }
    
    [arr[i + 1], arr[high]] = [arr[high], arr[i + 1]];  // 基准归位
    return i + 1;               // 返回基准位置
}
```

**关键点记忆：**

```
📍 i = low - 1  ← 为什么？因为一开始小于区是空的！
📍 先 i++ 再交换 ← 为什么？先腾位置再放东西！
📍 自己换自己 ← 正常！代码统一简洁，不影响结果
📍 最后交换 arr[i+1] 和 arr[high] ← 让基准到中间当分界线
```

**执行效果：**

```
分区前：[3, 1, 4, 1, 5, 9, 2, 6]
                  ↑
               pivot=6

分区后：[3, 1, 4, 1, 5, 2]  6  [9]
        ↑              ↑   ↑   ↑
      都小于 6      i+1  pivot 都大于 6
```

---

#### **2. 快速排序主函数 - 递归的力量**

```javascript
function quickSort(arr, low = 0, high = arr.length - 1) {
    if (low < high) {
        const pivotIndex = partition(arr, low, high);  // 分区
        
        quickSort(arr, low, pivotIndex - 1);   // 排左边
        quickSort(arr, pivotIndex + 1, high);  // 排右边
    }
    return arr;
}
```

**递归过程图解：**

```
第 1 层：[3,1,4,1,5,2,6,9] → pivot=6
              ↓
        [3,1,4,1,5,2] 和 [9]
        
第 2 层：[3,1,4,1,5,2] → pivot=2
              ↓
        [1,1] 和 [4,1,5]
        
第 3 层：继续分...

直到每堆只有 1 个元素 → 完成！✅
```

---

#### **3. 时间和空间复杂度**

```
⏱️ 时间复杂度：
- 最好情况：O(n log n) - 每次都完美平分
- 平均情况：O(n log n) - 随机数据
- 最坏情况：O(n²) - 已经有序时（可以随机选 pivot 避免）

💾 空间复杂度：O(log n) - 递归调用栈的深度

🔄 稳定性：❌ 不稳定（相同元素可能被换乱）
```

## 🃏 三、插入排序快速回顾（3 分钟）

### **核心思想：**

> **就像整理扑克牌，摸一张新牌，插到合适位置**

### **代码实现：**

```javascript
function insertionSort(arr) {
    for(let i = 1; i < arr.length; i++){
        let value = arr[i]      // 要插入的牌
        let j = i - 1
        
        // 把比 value 大的牌往后挪
        while(j >= 0 && arr[j] > value){
            arr[j + 1] = arr[j]
            j--
        }
        
        arr[j + 1] = value      // 插入到正确位置
    }
    return arr
}
```

### **关键点：**

```
✅ 从第 2 个元素开始（i=1）
✅ 前面的默认已排序
✅ 找到位置就插进去
✅ 后面的元素整体后移

时间复杂度：O(n²)
空间复杂度：O(1)
稳定性：✅ 稳定
```

## 🔀 四、归并排序快速回顾（3 分钟）

### **核心思想：**

> **先拆分到最小，再合并有序的**

### **代码实现：**

```javascript
function mergeSort(arr) {
    if(arr.length <= 1) return arr  // 只有一个元素，已有序
    
    let mid = Math.floor(arr.length/2)
    let left = mergeSort(arr.slice(0, mid))   // 拆左半
    let right = mergeSort(arr.slice(mid))     // 拆右半
    
    return merge(left, right)  // 合并两个有序数组
}

function merge(left, right) {
    let result = []
    let i = 0, j = 0
    
    // 两个有序数组合并成一个
    while(i < left.length && j < right.length){
        if(left[i] < right[j]){
            result.push(left[i])
            i++
        } else {
            result.push(right[j])
            j++
        }
    }
    
    // 把剩余的加到后面
    return [...result, ...left.slice(i), ...right.slice(j)]
}
```

### **关键点：**

```
✅ 分治思想的典型应用
✅ 先递归拆分，再合并
✅ 合并时保持有序
✅ 需要额外 O(n) 空间

时间复杂度：O(n log n) - 所有情况都是
空间复杂度：O(n)
稳定性：✅ 稳定
```

## 🎯 六、考试/面试必考点（3 分钟）

### **考点 1：手写快速排序**

```javascript
// 必须能默写出来！
function quickSort(arr, low = 0, high = arr.length - 1) {
    if (low < high) {
        const pivotIndex = partition(arr, low, high);
        quickSort(arr, low, pivotIndex - 1);
        quickSort(arr, pivotIndex + 1, high);
    }
    return arr;
}

function partition(arr, low, high) {
    const pivot = arr[high];
    let i = low - 1;
    
    for (let j = low; j < high; j++) {
        if (arr[j] < pivot) {
            i++;
            [arr[i], arr[j]] = [arr[j], arr[i]];
        }
    }
    
    [arr[i + 1], arr[high]] = [arr[high], arr[i + 1]];
    return i + 1;
}
```

**记忆口诀：**

> **选基准在最后，i 减一站外面。**  
> **j 扫描比大小，小就 i++ 来交换。**  
> **循环完换中间，返回 i+1 做分界。**  
> **左右递归继续排，完成排序很简单！**

---

### **考点 2：解释 partition 过程**

**标准答案模板：**

```
1. 选择最后一个元素作为基准（pivot）
2. 初始化 i = low - 1，表示小于区的右边界
3. 用 j 从 low 到 high-1 扫描
4. 如果 arr[j] < pivot，说明应该放入小于区
   - i++ 扩展小于区
   - 交换 arr[i] 和 arr[j]
5. 循环结束后，交换 arr[i+1] 和 arr[high]
   - 让 pivot 到达最终位置
6. 返回 i+1，这就是 pivot 的索引
```

---

### **考点 3：为什么快速排序最快？**

**标准答案：**

```
1. 平均时间复杂度 O(n log n)，与归并排序相同

2. 但是：
   - 原地排序，不需要额外空间（归并需要 O(n)）
   - 缓存局部性好，CPU 喜欢连续访问
   - 常数因子小，实际运行更快

3. 实际应用：
   - C 标准库的 qsort()
   - Java Arrays.sort()（早期版本）
   - 是最常用的通用排序算法
```

---

### **考点 4：快速排序 vs 归并排序**

**对比要点：**

```
相同点：
- 都用分治思想
- 平均都是 O(n log n)

不同点：
- 快排原地排序，归并不是
- 快排不稳定，归并稳定
- 快排最坏 O(n²)，归并总是 O(n log n)
- 快排缓存友好，归并需要额外空间

选择建议：
- 一般情况：选快速排序
- 要求稳定：选归并排序
- 内存紧张：选快速排序
- 链表排序：选归并排序
```

```javascript
// 优化 1：随机选 pivot，避免最坏情况
function partitionRandom(arr, low, high) {
    const randomIndex = Math.floor(Math.random() * (high - low + 1)) + low;
    [arr[randomIndex], arr[high]] = [arr[high], arr[randomIndex]];
    return partition(arr, low, high);
}

function quickSortOptimized(arr, low, high) {
    if (high - low < 10) {
        insertionSort(arr, low, high);
        return;
    }
    // ... 正常快速排序
}

// 优化 3：三路快排（处理大量重复元素）
function quickSort3Way(arr, low, high) {
    if (low >= high) return;
    
    let lt = low, gt = high;
    const pivot = arr[low];
    let i = low + 1;
    
    while (i <= gt) {
        if (arr[i] < pivot) {
            [arr[lt], arr[i]] = [arr[i], arr[lt]];
            lt++; i++;
        } else if (arr[i] > pivot) {
            [arr[i], arr[gt]] = [arr[gt], arr[i]];
            gt--;
        } else {
            i++;
        }
    }
    
    quickSort3Way(arr, low, lt - 1);
    quickSort3Way(arr, gt + 1, high);
}
```

### **快速排序速查表**

```javascript
// 10 秒钟回顾
function quickSort(arr, low = 0, high = arr.length - 1) {
    if (low < high) {
        let p = partition(arr, low, high);
        quickSort(arr, low, p - 1);
        quickSort(arr, p + 1, high);
    }
    return arr;
}

function partition(arr, low, high) {
    let pivot = arr[high], i = low - 1;
    for (let j = low; j < high; j++)
        if (arr[j] < pivot)
            [arr[++i], arr[j]] = [arr[j], arr[i]];
    [arr[i + 1], arr[high]] = [arr[high], arr[i + 1]];
    return i + 1;
}
```

```
快排：O(n log n) 平均，O(n²) 最坏，O(log n) 空间
归并：O(n log n) 总是，O(n) 空间
插入：O(n²) 平均，O(n) 最好，O(1) 空间
```
