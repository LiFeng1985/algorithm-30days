## 😓 二、为什么"还是写不对"？

### **常见原因分析：**

```
❌ 原因 1：看得懂 ≠ 写得出来
   - 看教程觉得懂了（输入）
   - 自己动手就卡壳（输出）
   - 这是正常的！需要刻意练习

❌ 原因 2：一次写太多，没有分解
   - 想一口气写完整个算法
   - 结果到处是 bug，越改越乱
   - 应该分解成小步骤，逐个击破

❌ 原因 3：缺少"肌肉记忆"训练
   - 算法需要像弹钢琴一样的肌肉记忆
   - 需要反复练习同样的代码模式
   - 建议：同一个算法写 5-10 遍

❌ 原因 4：没有及时复习
   - 今天学的明天就忘了一半
   - 建议：每天花 10 分钟复习昨天的内容

❌ 原因 5：追求完美，害怕犯错
   - 写一行就想对不对
   - 结果畏手畏脚，进度很慢
   - 建议：先写出来，再调试修改
```

## 💪 三、解决方案：渐进式训练法

### **第 1 阶段：抄代码（1-2 天）**

```javascript
// 不要觉得丢人，抄代码是最快的学习方式！

步骤：
1. 打开正确的代码（比如 sort6.txt）
2. 照着抄一遍到空白文件
3. 边抄边理解每一行的作用
4. 抄完后运行测试，确保正确

好处：
✅ 建立代码的"手感"
✅ 熟悉语法和结构
✅ 不会打击自信心
```

---

### **第 2 阶段：填空练习（1-2 天）**

```javascript
// 我给你留空的代码，你来填空

例如插入排序：
function insertionSort(arr){
    for(let i = 1; i < arr.length; i++){
        let j = ______;          // 提示：i 的前一个位置
        let value = ______;      // 提示：要插入的值
        
        while(j >= 0 && value < arr[j]){
            ______ = arr[j];     // 提示：后移操作
            j--;
        }
        
        ______ = value;          // 提示：插入到正确位置
    }
    return arr;
}

好处：
✅ 锻炼关键逻辑的记忆
✅ 有提示，不会太难
✅ 逐步建立信心
```

---

### **第 3 阶段：默写（2-3 天）**

```javascript
// 不看答案，自己默写

步骤：
1. 先看一遍正确的代码（5 分钟）
2. 关掉参考答案
3. 凭记忆默写整个算法
4. 写不出来就空着，最后再对照答案
5. 对比找出差距，重点记忆

好处：
✅ 真正检验是否掌握
✅ 强化长期记忆
✅ 发现薄弱环节
```

---

### **第 4 阶段：变式练习（2-3 天）**

```javascript
// 改变条件，测试是否真正理解

例如：
1. 把升序改成降序
2. 把数组改成从大到小排序
3. 添加打印语句，观察执行过程
4. 处理特殊情况（负数、小数等）

好处：
✅ 深入理解原理
✅ 培养举一反三能力
✅ 不再死记硬背
```

## 🎨 四、具体学习计划（以插入排序为例）

## 🔧 五、实用小技巧

### **技巧 1：分块记忆**

```javascript
不要试图一次性记住所有代码！

插入排序可以分成 3 块：
块 1: for 循环（遍历数组）
块 2: while 循环（找位置 + 后移）
块 3: 插入操作（放入正确位置）

每次只专注 memorize 一块，最后组合起来。
```

---

### **技巧 2：口诀记忆**

```javascript
给算法编一个口诀：

插入排序口诀：
"从第二个开始，摸一张牌
往前找位置，大的往后挪
找到合适位，把手中的牌放下"

快速排序口诀：
"选个基准值，分成两半
小的放左边，大的放右边
左右分别再来一遍"

朗朗上口，容易记忆！
```

---

### **技巧 3：画图辅助**

```javascript
不要只在脑子里想，动手画！

例如插入排序 [5, 2, 8, 1]：

初始：[5, 2, 8, 1]
       ↑
       i=1, value=2

比较：2 < 5，5 后移
      [5, 5, 8, 1]
       ↑
       j=0

插入：arr[0] = 2
      [2, 5, 8, 1]
       
继续下一轮...

画图让抽象的逻辑变得具体！
```

---

### **技巧 4：录音自听**

```javascript
1. 把自己讲解算法的声音录下来
2. 通勤、走路、做家务时听
3. 重复听，加深印象
4. 相当于随身带了个私教

示例讲解词：
"插入排序就像整理扑克牌..."
"首先，我们从第二张牌开始..."
"然后，我们往前找位置..."
```

### **2. 调试练习**

```javascript

function insertionSort(arr){
    for(let i = 1; i < arr.length; i++){
        let j = i;  // ❌ 错在哪里？
        let value = arr[i];
        while(j >= 0 && value < arr[j]){
            arr[j] = arr[j+1];  // ❌ 错在哪里？
```
