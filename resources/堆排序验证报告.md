### **2. heapify 函数 ✅**
```javascript
function heapify(arr, n, i) {
    let largest = i;
    let left = 2 * i + 1;
    let right = 2 * i + 2;
    
    if (left < n && arr[left] > arr[largest]) {
        largest = left;
    }
    
    if (right < n && arr[right] > arr[largest]) {
        largest = right;
    }
    
    if (largest !== i) {
        swap(arr, i, largest);
        heapify(arr, n, largest);
    }
}
```
**评价：**
- ✅ 正确计算左右孩子索引
- ✅ 正确比较并更新最大值索引
- ✅ 正确递归调整子树
- ✅ 变量命名清晰（largest, left, right）

---

### **3. heapSort 主函数 ✅**
```javascript
function heapSort(arr) {
    const n = arr.length;
    
    // 步骤 1：构建最大堆
    for (let i = Math.floor(n / 2) - 1; i >= 0; i--) {
        heapify(arr, n, i);
    }
    
    // 步骤 2：依次取出堆顶元素
    for (let i = n - 1; i > 0; i--) {
        swap(arr, 0, i);
        heapify(arr, i, 0);
    }
    
    return arr;
}
```
**评价：**
- ✅ 正确从最后一个非叶子节点开始构建堆
- ✅ 正确地将堆顶元素与末尾元素交换
- ✅ 正确地重新调整剩余元素
- ✅ 循环边界条件正确
