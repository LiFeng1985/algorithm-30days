## 🔄 方法一：迭代法（三指针翻转）

### 核心思想

通过三个指针的配合，逐个改变节点的指向，就像"排队的人向后转"。

### 三个指针的角色

```
prev (或 low)   - 前驱指针：指向已反转部分的头节点（初始为 NULL）
current (或 pro)- 当前指针：指向待反转部分的头节点（初始为 head）
nextTemp (或 temp) - 临时指针：保存下一个节点（防止丢失）
```

### 详细步骤图解

假设链表：`20 → 10 → 30 → 40 → NULL`

```
【初始状态】
head → 20 → 10 → 30 → 40 → NULL
       ↑
    current
prev = null


【第 1 次循环】处理节点 20
步骤 1: nextTemp = current.next  (保存节点 10)
步骤 2: current.next = prev      (20 指向 null)
步骤 3: prev = current           (prev 移动到 20)
        current = nextTemp       (current 移动到 10)

结果：null ← 20   10 → 30 → 40
              ↑    ↑
            prev  current


【第 2 次循环】处理节点 10
步骤 1: nextTemp = current.next  (保存节点 30)
步骤 2: current.next = prev      (10 指向 20)
步骤 3: prev = current           (prev 移动到 10)
        current = nextTemp       (current 移动到 30)

结果：null ← 20 ← 10   30 → 40
                   ↑    ↑
                 prev  current


【第 3 次循环】处理节点 30
步骤 1: nextTemp = current.next  (保存节点 40)
步骤 2: current.next = prev      (30 指向 10)
步骤 3: prev = current           (prev 移动到 30)
        current = nextTemp       (current 移动到 40)

结果：null ← 20 ← 10 ← 30   40
                        ↑    ↑
                      prev  current


【第 4 次循环】处理节点 40
步骤 1: nextTemp = current.next  (保存 null)
步骤 2: current.next = prev      (40 指向 30)
步骤 3: prev = current           (prev 移动到 40)
        current = nextTemp       (current 移动到 null)

结果：null ← 20 ← 10 ← 30 ← 40
                             ↑
                           prev


【循环结束】current = null
this.head = prev  (head 指向 40，成为新的头节点)

最终结果：40 → 30 → 10 → 20 → NULL
```

### 代码实现（迭代法）

```javascript
// 你的代码版本 - 完全正确！
reverse() {
    let prev = null;           // 前驱指针，初始为 null
    let current = this.head;   // 当前指针，从头节点开始
    let tail = this.head;      // 保存原来的头节点（它会变成尾节点）
    
    while (current != null) {
        // 步骤 1：先保存下一个节点（不然修改 next 后会丢失）
        let nextTemp = current.next;
        
        // 步骤 2：反转指针（当前节点指向前一个节点）
        current.next = prev;
        
        // 步骤 3：移动指针（两个指针都向前走一步）
        prev = current;        // prev 移动到当前节点
        current = nextTemp;    // current 移动到下一个节点
    }
    
    this.head = prev;          // prev 最后指向原链表的尾节点，成为新 head
    this.tail = tail;          // 原来的 head 成为新的 tail
}
```

```javascript
// 代码随想录版本 - 变量名不同，逻辑完全一样
function reverseList(head) {
    if (head === null || head.next === null) {
        return head;
    }
    
    let low = null;    // 相当于 prev
    let pro = head;    // 相当于 current
    
    while (pro !== null) {
        let temp = pro;       // 保存当前节点
        pro = pro.next;       // pro 向后移动
        temp.next = low;      // 反转当前节点指向
        low = temp;           // low 向后移动
    }
    
    return low;  // low 成为新的头节点
}
```

### 记忆口诀

> **"一存二移三反转"**
> 
> 1. **存**：保存下一个节点（nextTemp）
> 2. **反转**：当前节点指向前一个（current.next = prev）
> 3. **移**：两个指针都向前移动（prev 和 current）

## 🌀 方法二：递归法（从后往前）

### 核心思想

**"先递后归"** - 先递归到链表尾部，再从后往前逐个反转节点指向。

### 递归过程图解

```
【原链表】
1 → 2 → 3 → 4 → 5 → NULL

【递归调用过程】("递"的过程，一直深入到尾部)
┌──────────────────────────────────────┐
│ reverseList(1)                       │
│ └─→ reverseList(2)                   │
│     └─→ reverseList(3)               │
│         └─→ reverseList(4)           │
│             └─→ reverseList(5)       │ ← 基准情况：到达尾部
│                 return 5 (返回最后一个节点)
└──────────────────────────────────────┘

【回溯反转过程】("归"的过程，从后往前反转)

reverseList(5) 返回 newHead = 5
  ↓
reverseList(4):
  - head = 4
  - newHead = reverseList(5) = 5
  - head.next.next = head  →  5.next = 4 ✅ (让 5 指向 4)
  - head.next = null       →  4.next = NULL (防止环)
  - return newHead = 5
  
  此时：1 → 2 → 3 → 4 ← 5
                    ↑    ↑
                   head  newHead

reverseList(3):
  - head = 3
  - newHead = reverseList(4) = 5
  - head.next.next = head  →  4.next = 3 ✅ (让 4 指向 3)
  - head.next = null       →  3.next = NULL
  - return newHead = 5
  
  此时：1 → 2 → 3 ← 4 ← 5
              ↑    ↑
             head  newHead

reverseList(2):
  - head = 2
  - newHead = reverseList(3) = 5
  - head.next.next = head  →  3.next = 2 ✅ (让 3 指向 2)
  - head.next = null       →  2.next = NULL
  - return newHead = 5
  
  此时：1 → 2 ← 3 ← 4 ← 5
          ↑    ↑
         head  newHead

reverseList(1):
  - head = 1
  - newHead = reverseList(2) = 5
  - head.next.next = head  →  2.next = 1 ✅ (让 2 指向 1)
  - head.next = null       →  1.next = NULL
  - return newHead = 5
  
  此时：1 ← 2 ← 3 ← 4 ← 5
       ↑                 ↑
      head             newHead

【最终结果】
newHead = 5 (新链表的头节点)
5 → 4 → 3 → 2 → 1 → NULL
```

### 代码实现（递归法）

```javascript
function reverseList(head) {
    // 基准条件：到达链表尾部或空链表
    if (head === null || head.next === null) {
        return head;  // 返回最后一个节点作为新头
    }
    
    // 递归处理剩余节点
    // newHead 始终是原链表的最后一个节点（新链表的头）
    let newHead = reverseList(head.next);
    
    // 关键一步：让下一个节点指向当前节点（反转）
    head.next.next = head;
    
    // 断开当前节点的指向，防止形成环
    head.next = null;
    
    // 始终返回原链表的最后一个节点（新链表的头）
    return newHead;
}
```

### 递归理解要点

1. **基准情况**：`head === null || head.next === null`  
   到达最后一个节点时，直接返回该节点

2. **递归假设**：假设 `head.next` 之后的部分已经反转好了

3. **关键代码**：`head.next.next = head`  
   让下一个节点指向当前节点（实现反转）

4. **断链防环**：`head.next = null`  
   防止形成循环链表

### 记忆口诀

> **"先递到底，再归回来反转"**
> 
> - **递**：一直递归到链表尾部
> - **归**：回溯时逐个反转节点指向

## 💡 为什么迭代法更推荐？

### 优势对比

```
【迭代法】✅
- 空间复杂度 O(1)，只用了几个指针
- 没有函数调用栈开销
- 逻辑直观，容易理解
- 面试首选方案

【递归法】⚠️
- 空间复杂度 O(n)，每次递归占用栈空间
- 链表很长时可能栈溢出
- 代码简洁但理解抽象
- 体现算法思维，可作为加分项
```

### 实际测试

假设链表有 10000 个节点：

```
迭代法：
- 内存占用：几个指针变量（忽略不计）
- 执行效率：高 ✅

递归法：
- 内存占用：10000 层函数调用栈
- 执行效率：较低
- 风险：可能导致栈溢出 ❌
```

## 🎯 常见面试变形题

### 1. 反转链表的前 N 个节点

```javascript
// 只反转前 n 个节点，后面的保持不变
function reverseN(head, n) {
    let successor = null;  // 记录第 n+1 个节点
    
    function reverse(head, n) {
        if (n === 1) {
            successor = head.next;  // 记录后继节点
            return head;
        }
        
        let last = reverse(head.next, n - 1);
        head.next.next = head;
        head.next = successor;  // 连接后继节点
        return last;
    }
    
    return reverse(head, n);
}
```

### 2. 反转链表的一部分

```javascript
// 反转从位置 m 到 n 的节点
function reverseBetween(head, m, n) {
    // 找到第 m-1 个节点
    let dummy = new ListNode(0);
    dummy.next = head;
    let prev = dummy;
    
    for (let i = 0; i < m - 1; i++) {
        prev = prev.next;
    }
    
    // 从第 m 个节点开始反转
    let cur = prev.next;
    for (let i = 0; i < n - m; i++) {
        let next = cur.next;
        cur.next = next.next;
        next.next = prev.next;
        prev.next = next;
    }
    
    return dummy.next;
}
```

### 3. K 个一组反转链表（LeetCode 25）

```javascript
// 每 k 个节点为一组进行反转
function reverseKGroup(head, k) {
    if (head === null) return null;
    
    let a = head, b = head;
    
    // 判断是否有 k 个节点
    for (let i = 0; i < k; i++) {
        if (b === null) return head;  // 不足 k 个，不反转
        b = b.next;
    }
    
    // 反转前 k 个节点
    let newHead = reverseAtoB(a, b);
    
    // 递归处理后续节点
    a.next = reverseKGroup(b, k);
    
    return newHead;
}

// 反转区间 [a, b) 的节点
function reverseAtoB(a, b) {
    let prev = null, cur = a;
    while (cur !== b) {
        let next = cur.next;
        cur.next = prev;
        prev = cur;
        cur = next;
    }
    return prev;
}
```

## 🎓 总结

### 核心要点

```
【迭代法】
- 三指针：prev, current, nextTemp
- 口诀："一存二移三反转"
- 空间 O(1)，时间 O(n)
- 面试首选 ✅

【递归法】
- 基准条件：head === null || head.next === null
- 关键：head.next.next = head
- 空间 O(n)，时间 O(n)
- 体现算法思维 ⭐
```

### 一句话总结

> **迭代法是"逐个击破"，递归法是"先深入敌后，再从后往前打"**
