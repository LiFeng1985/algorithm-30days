## 💡 二、红黑树的本质

### **一句话理解：**

> **红黑树 = 会自动调整的平衡树**

### **核心思想：**

```
就像玩魔方：
- 如果某层太长了 → 转一下（旋转）
- 如果颜色不对了 → 换个色（变色）

通过旋转和变色，让树保持平衡！
```

---

## 🎨 三、5 条规则（简化版）

### **规则其实很简单：**

```
1. 节点要么红色，要么黑色（就两种颜色）

2. 根是黑色（老大必须是黑色）

3. 红色节点不能有红色孩子（红不相邻）
   🔴
  /  \
⚫    ⚫  ✅

🔴
/  \
🔴  🔴  ❌ 不行！

4. 从根到叶子的每条路径，黑色节点数相同
   （这个最难理解，先记住就行）

5. 叶子（空的）是黑色
   （这个不用管，代码里不画出来）
```

## 🔄 四、两个核心操作

### **操作 1：变色**

```javascript
// 就是把红色变黑色，黑色变红色
node.color = !node.color

// 就像换衣服一样简单
```

---

### **操作 2：旋转**

```
左旋：把右边的提起来

    5              7
   / \            / \
  A   7    =>    5   C
     / \        / \
    B   C      A   B
    
5 下来，7 上去
```

```
右旋：把左边的提起来

    7              5
   / \            / \
  5   C    =>    A   7
 / \                 / \
A   B               B   C

7 下来，5 上去
```

**记忆技巧：**

> "哪边重就往相反方向旋"
> - 右边重（右子树大）→ 左旋
> - 左边重（左子树大）→ 右旋

## 🎯 五、插入操作（重点中的重点）

### **插入分三步：**

```
第 1 步：像普通二叉树那样插入

第 2 步：新节点涂成红色
        （为什么？因为红色影响小）

第 3 步：如果违反规则，就调整
```

---

### **只有 3 种情况：**

```
假设你插入了一个新节点 N

情况 1：叔叔是红色
解决：爸爸和叔叔变黑，爷爷变红

情况 2：叔叔是黑色，你在左边
解决：右旋爷爷，爸爸变黑，爷爷变红

情况 3：叔叔是黑色，你在右边
解决：先左旋爸爸（变成情况 2），再按情况 2 处理
```

**图解情况 2（最常见）：**

```
插入前：
    ⚫ 爷爷
   /   \
 🔴 爸爸  ⚫ 叔叔
 /
🔴 你 ← 刚插入

问题：爸爸和你都是红色！

解决：
    🔴 爸爸
   /     \
 🔴 你    ⚫ 爷爷
         /   \
       ...   ⚫ 叔叔

完成！✅
```

## 💻 六、代码实现（最简版）

### **Node 类：**

```javascript
class Node {
    constructor(data) {
        this.data = data;
        this.color = 'RED';  // 新节点都是红色
        this.left = null;
        this.right = null;
    }
}
```

---

### **最简单的插入代码：**

```javascript
class RedBlackTree {
    constructor() {
        this.root = null;
    }
    
    insert(data) {
        this.root = this._insert(this.root, data);
        this.root.color = 'BLACK';  // 根变黑
    }
    
    _insert(node, data) {
        if (!node) return new Node(data);
        
        if (data < node.data) {
            node.left = this._insert(node.left, data);
        } else {
            node.right = this._insert(node.right, data);
        }
        
        return this._fix(node);  // 修复
    }
    
    _fix(node) {
        // 情况 1：右边红，左边黑 → 左旋
        if (this._isRed(node.right) && !this._isRed(node.left)) {
            node = this._rotateLeft(node);
        }
        
        // 情况 2：左边红，左左也红 → 右旋
        if (this._isRed(node.left) && this._isRed(node.left.left)) {
            node = this._rotateRight(node);
        }
        
        // 情况 3：左右都红 → 变色
        if (this._isRed(node.left) && this._isRed(node.right)) {
            node.color = !node.color;
            node.left.color = !node.left.color;
            node.right.color = !node.right.color;
        }
        
        return node;
    }
    
    _rotateLeft(node) {
        const newRoot = node.right;
        node.right = newRoot.left;
        newRoot.left = node;
        newRoot.color = node.color;
        node.color = 'RED';
        return newRoot;
    }
    
    _rotateRight(node) {
        const newRoot = node.left;
        node.left = newRoot.right;
        newRoot.right = node;
        newRoot.color = node.color;
        node.color = 'RED';
        return newRoot;
    }
    
    _isRed(node) {
        return node && node.color === 'RED';
    }
}
```
