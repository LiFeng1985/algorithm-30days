## 🎯 使用建议

### 学生使用方法

```
每天学习流程：
1. 先看【学习检查清单】- 了解今天要学什么
2. 再学【主教程】- 详细学习内容
3. 对照【思维导图】- 构建知识框架
4. 完成【练习题】- 检验学习效果
5. 看【答案详解】- 理解解题思路
6. 可选：观看视频讲解（如有）
```

### 老师使用方法

```
教学流程：
1. 用【视频脚本】录制预习视频
2. 发【学习检查清单】给学生
3. 课堂讲解主教程内容
4. 用【思维导图】总结
5. 布置【练习题】作为作业
6. 提供【答案详解】供参考
```
